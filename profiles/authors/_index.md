# 作者风格档案索引

> 每个作者一个 `.md` 文件，按作者名命名。调用时 `ls` 列出可用档案，按需读取。

## 档案结构

每个作者档案包含（详见 `author-profiling` skill）：

1. 叙述者声音肖像 — 语调范围 / 幽默机制 / 情感距离控制
2. 情节设计模式 — 驱动类型 / 冲突偏好 / 高潮模式 / 情节模板
3. 关系设计模式 — 核心关系类型 / 关系组合模板 / 推进节奏
4. 常用技法清单 — 开头/桥段/断章/事件展开/特殊技法
5. 世界观设计风格 — 构建方式 / 规则密度 / 社会结构
6. 大纲模板 — 全书结构 / 章节模板 / 节奏模板 / 信息释放节奏
7. 语言指纹 — 句长分布 / 词汇偏好 / 标点特征 / 比喻密度

## 使用场景

- **写作流启发式交谈**：提供"如果是 XX 作者会如何写"视角
- **改编流风格对标**：L1（文风移植）时作为目标风格约束
- **「墨」写作**：参照作者档案的叙述者声音和语言指纹
- **「铃」风格把关**：将作者档案的语言指纹作为额外检查维度
