# 感官锚定写作工具 — 方法三 AVG 脚本模式协议

> **调用条件**：`output_format="script"` 时激活
> **格式规范**：`framework/guides/avg-script-format.md` + `framework/guides/audiobook-script-guide.md`
> **格式速查**：`framework/templates/_script-format-spec.md`
> **立绘规格**：`framework/templates/characters/_sprite-spec.md`

## 一、逐场景协议

```
0. [场景锚定] 写【场景：{地点}·{时间}】

1. [视听环境] 【BGM：曲风/情绪】 + 【bg：背景ID】 + 旁白建场（空间/光影/环境细节）

2. [角色登场] 首次出场 → 【立绘：角色 表情 位置 滑入】
   → 已在场景中？跳过（除非表情变化）

3. 逐句生成——每叙事单元判断类型：
   a. 台词 → 角色名 [表情]：「台词」+ ⏱ N.Ns
      → max(0.5, 纯中文字数/3.5 + 标点附加）
      → 句末。！？+0.3s / ……+0.5s / ，、+0.15s / 单字≥1.0s
   b. 演出注 → 对话下方缩进（注：动作/语气/微表情）
      → 写看得见的肢体动作和与嘴上说的矛盾的微表情
      → 不写抽象心理；不重述对话内容
   c. 表情变 → 先【立绘：角色 新表情】+ 下句对话 [新表情] 同步更新
   d. 音效 → 【音效：描述】——触发动作前
   e. 旁白 → 【旁白】...【/旁白】
      → 可写：场景环境 / 主角内心独白 / 时间流逝
      → 不写抽象评价
   f. CG → 【CG：名称 | 触发时机】
      → 🎨 生图提示词：日式动画CG风格，16:9横版。{角色名}——{动作/表情/服装}。{场景环境+光影+构图}
      → 用于情感高潮/初遇/章末名场面定格
   g. 停顿 → 【bgm 静默 N秒】或【停顿 N秒】
   h. 检定 → 【检定：技能 | 难度：NN | 结果】
      → 检定演出注：骰子类型/视觉/音效/情绪
      → 仅跑团 Replay VN

4. [角色退场] → 【立绘：角色 淡出】

5. [场景收束] 场景结束 → 下一个场景从步骤 0 开始

6. [章末汇总] 所有场景写完：
   a. 配音时长汇总表
   b. 背景生图提示词索引——本章所有 bg ID 对应的 GPT 生图提示词
   c. CG 生图提示词索引——本章所有 CG 对应的 GPT 生图提示词
```

## 二、AVG 脚本护栏

1. **角色名 [表情] 直接前缀**——不用散文对话标签
2. **每句对白必带 `[表情]` + `⏱ N.Ns`**
3. **情绪变化** = `【立绘：角色 新表情】` + 下句对白 `[新表情]` 同步
4. **演出注写"看得见的"**——肢体动作/微表情/环境互动；不解释动机
5. **旁白内心独白可保留**（AVG 旁白承担内心叙事）
6. **CG 提示词仅写角色姓名**——样貌从立绘规格卡获取
7. **背景提示词含**：场景描述+光影+关键视觉元素+构图视角
8. **项目类型决定块类型**：
   - 跑团 Replay VN 不用选择块/跳转/flag
   - 标准 AVG 不用检定块
