# 7 维作者风格分析协议（外移参考）

> 合并自原 `author-profiling` Skill。被 `qing-novelist` Skill「作者分析模式」Step 4 按需读取。
> **核心信念**：从作品提取作者，而非从作者标签推导作品。所有分析基于文本证据，不做"因为 TA 是 XX 类型作家所以一定 XX"的预设。

---

## 一、分析流程骨架

```
第一步：分卷分段 → 每卷一个独立提取单元
第二步：逐卷全量 7 维提取（不抽样，定性提取覆盖全部章节）
第三步：跨卷对比 + 风格演化分析
第四步：形成作者档案
第五步：技法入库
第六步：更新索引
```

---

## 二、第一步：分卷分段

1. 按自然分卷/分篇切割为独立提取单元（无分卷则每 10-15 章一个）
2. 每个单元独立分析，最后做跨卷对比
3. 长篇（>100 章）采用"全量提取前 3 卷 + 后续每卷抽样 30%"——前 3 卷必须全量

**每卷中间结果**：`profiles/authors/_tmp/{作者名}-V{卷号}.md`

---

## 三、第二步：逐卷全量 7 维提取

> **关键规则**：定性提取（情节设计/关系设计/技法）必须覆盖全部章节。抽样只适用于统计类数据（句长/词汇频率等）。
> 每条提取信息标注 `[卷X-ChY]` 位置标签。

### 维度 1：叙述者风格

| 提取项 | 说明 |
|--------|------|
| 叙述者身份 | 第几人称？角色/旁观者/上帝视角？与读者的关系（亲密/疏离/戏弄/教导）？ |
| 叙述者声音 | 5-8 句具体描述"这个叙述者听起来像什么样的人"——语气/态度/幽默感/毒舌程度/情感温度 |
| 语调范围 | 上限（最激烈时）+ 下限（最冷峻时）+ 常态 |
| 幽默机制 | 来源（叙述者吐槽/角色对话反差/情境荒诞/概念置换/…）+ 2-3 个代表性片段 |
| 情感距离控制 | 何时拉近/拉远？距离切换的信号？ |
| 议论/评价习惯 | 冷嘲/感慨/哲理/克制不评？ |
| 不可靠性 | 隐瞒信息/选择性记忆/自我欺骗/… |

**输出**：200-400 字"叙述者声音肖像" + 语调范围图（文字描述）

### 维度 2：情节设计模式

| 提取项 | 说明 |
|--------|------|
| 情节驱动类型 | 角色驱动 X% / 事件驱动 Y% |
| 冲突结构偏好 | 人与人/人与制度/人与自己/人与自然——各类型比例 |
| 高潮设计模式 | 信息揭露型/战斗决出型/角色选择型/多线汇集型 + 2-3 个示例 |
| 伏笔密度与回收周期 | 每章平均几个？同章/3 章内/跨卷？ |
| 转折偏好 | 身份/关系/信息/处境反转 + 频率 |
| 结局倾向 | 已完结→结局类型 / 未完→每卷收束方式 |
| 巧合容忍度 | 巧合使用程度 + 如何"合理化" |
| 情节模板 | 识别可复用模板——如"主角发现秘密→被利用→反击→揭露更大阴谋" |

**输出**：情节设计模式卡 + 可复用模板列表

### 维度 3：关系设计模式

| 提取项 | 说明 |
|--------|------|
| 核心关系类型 | 恋爱/友情/敌对/师徒/家庭/… |
| 关系起点偏好 | 意外相遇/被迫共处/长期暗恋/敌人变朋友/… |
| 关系推进节奏 | 从建立到质变多少章？节奏标记？ |
| 关系障碍设计 | 外在/内在/身份差距/误解/三角/… |
| 对话中的关系显现 | 潜台词密度？专属称呼/内部笑话/默契行为频率？ |
| 关系组合模板 | 反复出现的关系组合——如"吐槽男主+毒舌女主+天降对手+败犬青梅" |

**输出**：关系设计模式卡 + 关系组合模板

### 维度 4：常用技法清单

> **每种技法标注**：技法名称 + 功能描述 + 2-3 个原文示例 + 适用场景 + 使用频率（每 X 章一次）

| 提取项 | 说明 |
|--------|------|
| 开头方式 | 对话钩子/目击意外/高动作/独白氛围/宣言概念/概念置换/其他 + 频率 |
| 桥段模式 | 反复使用的场景桥段结构——提取并命名 |
| 断章方式 | 章节收束偏好 + 频率 |
| 事件展开 | 推进事件的方式偏好 |
| 对话技法 | 节奏/潜台词密度/信息嵌入/标签风格 |
| 特殊技法 | 该作者独有的、其他作品罕见的技法 |

**入库格式**（写入 `framework/templates/technique-library.md`）：

```markdown
### 技法名称
**来源**：[作者名] · [作品名]
**首次识别**：YYYY-MM-DD
[原有格式继续...]
```

### 维度 5：世界观设计风格

| 提取项 | 说明 |
|--------|------|
| 世界构建方式 | 角色探索发现/叙述者直接说明/对话碎片拼接/冲突暴露规则/… |
| 规则密度 | 硬规则（量化/系统性）vs 软规则（模糊/氛围性） |
| 规则与情节的关系 | 背景板 vs 驱动力（是否被利用/挑战/打破） |
| 日常细节密度 | 衣食住行/货币/社会运行描写详细程度 |
| 力量/能力体系设计 | 系统性/量化/成长路径/代价 |
| 社会结构设计 | 阶层/势力/组织复杂度？是否多层嵌套？ |
| 世界观的边界感 | 明确划定"不存在什么" vs 保持开放 |

**输出**：世界观设计风格卡

### 维度 6：大纲设计与篇幅结构

> 回答"他会如何设计这段篇章"——供「五更」规划大纲和「墨」分配篇幅时直接参照。

#### 6.1 全书篇幅结构

| 提取项 | 说明 |
|--------|------|
| 总字数 | 连载中则标注当前 + 预估总字数 |
| 分卷方式 | 数量/每卷章节数/字数范围/卷关系（独立弧/连续/渐强） |
| 卷内结构 | 统一结构模板？（如"日常→事件→升级→高潮→呼吸"） |

#### 6.2 章节篇幅分布（核心数据）

抽样统计（覆盖 30% 章节，均匀分布）：

| 提取项 | 说明 |
|--------|------|
| 章节字数范围 | 最短/最长/平均/中位数 |
| 字数分布形态 | <2000 / 2000-3000 / 3000-4000 / >4000 各占比 |
| 章节类型与字数 | 铺垫/高潮/过渡/日常/战斗 各多少字 |
| 每章场景数 | 平均 + 类型差异 |
| 场景平均字数 | 最短/最长/平均 |

#### 6.3 内容类型配比

| 提取项 | 说明 |
|--------|------|
| 对话占比 | 抽样统计 |
| 描写占比 | 环境/外貌/动作 |
| 内心独白占比 | |
| 叙述占比 | 推进时间/交代信息 |
| 不同章节类型的配比变化 | 战斗章对话少动作多/日常章对话多——各类型具体数字 |

#### 6.4 故事节拍密度

| 提取项 | 说明 |
|--------|------|
| 节拍密度 | 每 1000 字有意义事件数（角色状态变化/新信息/关系变化/冲突升级） |
| 信息释放密度曲线 | 均匀释放/集中爆发 |
| 伏笔铺设密度 | 每章平均 + 类型分布（世界观/角色/情节） |

#### 6.5 章节内部结构模板

| 章节类型 | 结构模板 | 字数分配 | 出现频率 |
|---------|---------|---------|---------|
| 铺垫/建立章 | 开头→场景1→场景2→场景3→结尾钩子 | 各场景字数 | X% |
| 高潮/解决章 | 开头（简短回顾）→ 高潮场景（长）→ 后果场景 → 结尾 | 高潮占60%+ | X% |
| 过渡/移动章 | 场景1（离开）→ 场景2（路上/信息）→ 场景3（到达/新状况） | 各场景均衡 | X% |
| 日常/呼吸章 | 日常场景1 → 日常场景2 → 一个异常细节 → 结尾 | 对话为主 | X% |
| 战斗/竞赛章 | 战前→战斗阶段1→阶段2→决胜→战后 | 战斗占70%+ | X% |

每种模板提取：开头方式 / 场景数及字数占比 / 高潮转折位置 / 结尾钩子类型

#### 6.6 弧线篇幅模板

| 弧线规模 | 章节数 | 总字数 | 典型结构 | 出现次数 |
|---------|--------|--------|---------|---------|
| 小弧（单事件） | X-Y | XX | | X |
| 中弧（篇章级） | X-Y | XX | | X |
| 大弧（卷级） | X-Y | XX | | X |

每种弧线：建立:发展:高潮:后果比例 / 内部节奏曲线 / 支线穿插

#### 6.7 章节开头与结尾模式

| 提取项 | 说明 |
|--------|------|
| 开头方式分布 | 回顾上章 / 环境切入 / 对话开场 / 内心独白 / 直接行动 — 各占 % |
| 结尾方式分布 | 悬念断章 / 情绪余韵 / 信息揭示 / 行动宣告 / 日常收束 — 各占 % |
| 开头-结尾配对 | 是否有固定配对？（如"环境切入→情绪余韵"是常态） |

#### 6.8 多线叙事篇幅（如适用）

| 提取项 | 说明 |
|--------|------|
| 各线字数占比 | A线/B线/C线 各占 % |
| 切换频率 | 平均每多少章切换？同章内切换？ |
| 各线章节字数差异 | |
| 线交汇时的篇幅处理 | 多线交汇章节字数是否明显增加？ |

### 维度 7：语言指纹

> 侧重"可量化的风格参数"

| 提取项 | 说明 |
|--------|------|
| 句长分布 | 抽样 500-1000 句，统计平均/中位数/标准差/短句(<10字)中句(10-30)长句(>30)比例 |
| 段落长度分布 | 平均段落句子数/极短(1句)/短(2-3句)/中(4-6句)比例 |
| 高频词汇 | 前 20 高频实词（动/形/名/副分别统计） |
| 标点特征 | 逗号密度/省略号/破折号/引号内对话占比 |
| 对话比例 | 占总字数比例/平均长度/标签位置偏好 |
| 比喻特征 | 密度（每千字几个）/来源域（自然/战争/食物/机械/…）/明喻vs暗喻比例 |
| 感官词汇分布 | 视觉/听觉/触觉/嗅觉/味觉比例 |
| 段落开头方式 | 名词/动词/连词/代词/… |

**输出**：语言指纹数据卡 + 与项目 `author-voice.md` 对比表

## 四、第三步：跨卷对比与风格演化分析

### 4.1 风格一致性 vs 演化

| 对比维度 | 对比内容 |
|---------|---------|
| 叙述者声音 | 各卷是否一致？微妙变化？ |
| 情节密度 | 每卷节拍密度变化 |
| 篇幅趋势 | 各卷平均章节字数变化 |
| 技法偏好转移 | 早期 vs 后期 |
| 关系设计演变 | 原型组合方式变化 |
| 世界观释放节奏 | 前密后疏/均匀/后期爆发 |
| 语言指纹稳定性 | 句长/词汇/比喻在各卷波动幅度 |

### 4.2 跨卷结构模式识别

| 对比维度 | 对比内容 |
|---------|---------|
| 卷级结构一致性 | 各卷是否遵循同一模板？ |
| 弧线规模分布 | 小/中/大弧在各卷分布——是否有规律？ |
| 章节类型分布 | 铺垫/高潮/日常比例稳定性 |
| 支线穿插模式 | 各卷出现频率和回收节奏 |

### 4.3 形成跨卷结论

- **稳定特征**（各卷一致）→ 标记"核心特征"
- **演化特征**（随卷变化）→ 标记"演化趋势"
- **波动特征**（各卷不一致）→ 标记"偶发手法"

## 八、作者档案使用方式

### 写作流使用

- 「五更」读取档案 → 在方向选项中增加"如果是 XX 作者"视角
- 「墨」写作时参考档案「叙述者声音肖像」+「语言指纹」对齐风格
- 「铃」/「琉璃」评审时，将作者档案语言指纹作为额外检查维度

### 改编流使用

- 改编阶段 1 指定对标作者时 → 档案作为改编目标风格
- 「墨」写作时同时对照原作画像（保留什么）+ 作者档案（写成什么风格）
- 「琉璃」评审时增加"风格对齐度"检查

### 多作者融合

- 用户希望融合多作者风格 → 「五更」在交谈中逐项确认（叙述声音对标 A / 情节设计对标 B / …）

### 指导写作模式（保持自己的声音，借技法）

| 维度 | 对标模式 | 指导模式 |
|------|---------|---------|
| 叙述者声音（§1） | 对齐参考作者 | **保持自己的声音**（author-voice.md 保护） |
| 情节设计（§2） | 模仿 | **诊断弱点 + 借用技法修补** |
| 关系设计（§3） | 套用模板 | **借鉴关系设计智慧** |
| 常用技法（§4） | 照搬 | **选取可迁移技法** |
| 世界观设计（§5） | 参照 | **可部分借鉴** |
| 大纲/篇幅（§6） | 套用模板 | **参考节奏智慧** |
| 语言指纹（§7） | 对齐 | **保持自己的指纹**（author-voice.md 保护） |

**指导模式工作流**：

```
1. 诊断：基于当前场景清单 + 情感目标，识别作者薄弱点
2. 匹配：从档案 §2/§3/§4/§6 匹配可填补薄弱点的具体技法
3. 建议：为每个薄弱点提供 1-2 个来自参考作者的具体技法建议
   （不是"你可以写得更有张力"，而是"风月在高潮章用电影化多线剪辑——每条线 200-400 字快速切换"）
4. 选择：用户选择采纳
5. 执笔：「墨」执笔时——叙述声音锚定 author-voice.md，技法执行参照选定的建议
6. 入库：采纳的技法写入 style-guide.md「特殊叙事技法」，标注 来源 + 采纳原因
```

## 九、轻量自我提取模式

**触发**：
- "提取我的写作风格" / "建立 author-voice.md"
- `pre-flight-check` 修复路径步骤 2（C3 author-voice.md 缺失时）
- 其他 Skill 调用 `Skill("qing-novelist", mode="分析", operation="extract-self")`

**与全量分析的区别**：
- 仅 §1 + §7 维
- 抽样 4-5 章（前 80 行 + 中 40 行 + 末 40 行）
- 不分析情节/关系/世界观/篇幅（这些是作者自己的创作特征）
- 不入库技法
- 产出 `{draft_dir}/author-voice.md`（含 frontmatter）

**执行流程**：

```
1. 确定抽样范围：
   - 有 draft_dir → 扫描 {draft_dir}/chapters/ 下的 .md
   - 降级 → 扫描 novel/chapters/ 下的 .md
   - 抽样策略：均匀分布取 4-5 章（< 10 章则全取）

2. 逐章轻量读取（每章前 80 + 中 40 + 末 40 行）：
   a. 叙述者声音特征（人称/语调/距离/幽默）
   b. 语言指纹（段长/句长/对话比例/感官/留白）

3. 综合写入：
   - 读 framework/templates/author-voice.md 获取完整模板
   - 填充 §一/§二/§三/§四/§七
   - §五 修炼/战斗 → 战斗章节则填充，否则"待补充"
   - §六 章节结构 → 基础填充（场景数/字数/开章方式）
   - 写入 {draft_dir}/author-voice.md（含 frontmatter）

4. 降级：
   - 章节总数 < 3 → ⚠️ "风格未稳定——建议写 5+ 章后重新提取"
   - 无战斗章节 → §五 "待补充"
   - 模板不存在 → 使用内嵌最小骨架（§一+§二+§三+§七）
```
