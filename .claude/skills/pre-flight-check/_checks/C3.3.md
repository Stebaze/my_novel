# C3.3：文件 sections frontmatter 检查

**条件**：target_chapter 已确定

## 检查范围

- `{draft_dir 或 novel/}` 根目录：`notes.md` `outline.md` `author-voice.md` `voice-bible.md` `thread-map.md` `character-arcs.md` `project-config.md`
- `characters/` 下当前章节出场角色档案
- `chapters/chapter-{N}.md`（当前章节）
- `world/rules.md` `world/setting.md` `world/timeline.md`

## 执行

1. `Read(file, limit=5)` → 首行为 `---` 且前 5 行包含 `sections:`
2. ✅ 存在 → 通过
3. ❌ 缺失 → ⚠️ 提醒："文件 X 缺少 sections frontmatter"
4. 累计 ≥ 3 个文件缺失 → 🟡 软阻断："{N} 个文件缺少 sections frontmatter，建议运行 file-manager ensure-frontmatter 补齐"

## 自动补齐

C3.3 检测到缺失 → 调 `Skill("file-manager", operation="ensure-frontmatter", target_dir="{draft_dir 或 novel/}")` → file-manager 从对应模板动态推导 frontmatter 并注入 → 不覆盖已有内容。
