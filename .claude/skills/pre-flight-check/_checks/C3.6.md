# C3.6：上一已写章节定位

**条件**：N > 1，C4/C7 的前置步骤

## 定位逻辑

1. `target = N - 1`
2. 读取 `chapters/chapter-{target}.md` → 检查「元数据」章节的「状态」字段
3. 状态 = "⬜ 纲要" / "⬜ 待写" → `target = target - 1`，重复步骤 2
4. 状态 = "📝 写作中" / "✅ 已完成" / "📦 已发布" → 使用 target
5. 未找到状态字段 → 判断文件内容是否含实际正文（非仅模板骨架）
   - 仅含模板骨架 + "（作者执笔）" → 未写 → target--
   - 含叙事正文（对话/描写/情节推进）→ 已写 → 使用 target

## 硬阻断

`target < N - 5`（向前追溯超过 5 章仍无已写章节）→ 🚫：

> 连续 5 章无已写正文，无法确定设定基线和角色状态。请先完成至少一章写作。

## 输出

`effective_prev_chapter = target`（供 C4/C7 使用）→ `target < N-1` 时标注 "Ch{N-1} 未写，C4/C7 以 Ch{target} 为实际前一章"。
