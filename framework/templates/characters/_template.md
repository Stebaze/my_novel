### 通用叙事原型（跨类型）

> 以下原型不限于恋爱轻小说，适用于奇幻/悬疑/冒险/正剧等所有类型。
> Agent 根据 `novel/project-config.md` 中的类型选择对应的原型池。

- [ ] **C1 英雄型**：背负使命 + 能力与责任匹配 + 核心考验是"在代价面前是否坚持"
- [ ] **C2 反英雄型**：手段灰色 + 目标可能正义也可能自私 + 魅力来自"他不守规则但他有底线"
- [ ] **C3 导师型**：经验丰富 + 有无法释怀的过去 + 在学生身上看到自己的影子
- [ ] **C4 变节者型**：曾在"另一边" + 改变阵营的原因不简单 + 新阵营的人不完全信任他
- [ ] **C5 探索者型**：被好奇心驱动 + 不满足于已知世界 + "真相"对他的吸引力大于一切
- [ ] **C6 守护者型**：保护某人/某物/某地是唯一目标 + 不善表达 + 可能过度保护
- [ ] **C7 复仇者型**：被过去伤害定义 + 复仇是目标但不是全部人格 + 何时放下是核心命题
- [ ] **C8 幸存者型**：经历了不该活下来的事 + 有隐秘的创伤 + 坚韧但并非无伤
- [ ] **C9 野心家型**：目标导向 + 愿意为此付出代价 + 可能跨越道德边界
- [ ] **C10 调和者型**：在冲突的各方之间周旋 + 不一定中立但追求平衡 + 核心矛盾是"当调和已经不可能时怎么办"

## 心理动力结构

> 操作框架：`framework/guides/jung-character-framework.md`。以荣格分析心理学为主线——人格面具与阴影的张力是角色核心戏剧冲突，自性化是成长弧线的底层引擎。

### 人格面具

> 角色在他人面前展示的社会身份。不是"真实的自己"——是"角色希望别人认为的自己"。

- **面具形象**：（在公众/不熟的人面前的样子——ta 希望别人怎么看待 ta）
- **面具行为规则**：（什么能做 / 什么不能说 / 什么情绪不能外露）
- **面具强度**：稳固（日常） / 松动（疲劳/情绪波动/面对特定人时出现的裂缝） / 卸下（独处/最信任的人面前/情绪崩溃时）

### 阴影

> 角色隐藏、否认或尚未意识到的心理内容——被压抑的欲望、不敢承认的恐惧、与面具形象冲突的特质。

- **阴影内容**：（ta 不敢承认或没有意识到的部分是什么？）
- **阴影侵入方式**：（阴影如何表达——投射到他人身上 / 情绪失控时溢出 / 独处时泄露的习惯 / 身体信号 / 梦境或幻觉）

### 面具维持策略

> 自我在面具受到威胁时自动启动的保护机制。参考 `framework/guides/jung-character-framework.md` §4。

- **主导策略**：（压抑 / 合理化 / 投射 / 反向形成 / 退行 / 升华——选 1-2 种，这是 ta 的"默认武器"）
- **触发场景**：（什么情境下 ta 会无意识启动这个策略）
- **外在表现**：（具体行为——不是心理描述，而是读者能看到/听到的动作和台词）

### 自性化阶段

> 角色整合人格面具与阴影、成为"完整的自己"的过程。参考 `framework/guides/jung-character-framework.md` §6。

- **当前针对的信念**：（ta 正在与什么信念"较劲"）
- **当前阶段**：① 无自觉默认 / ② 怀疑否认 / ③ 矛盾撕扯 / ④ 顿悟重构 / ⑤ 试错检验 / ⑥ 新平衡
- **阶段转变标志事件**：（发生了什么事让 ta 进入/离开这个阶段）

## 内在驱动（八维）

> 此节定义角色在具体场景中的"此刻动机→具体行动"推导链。参考 `framework/guides/jung-character-framework.md` §7。
> **Agent 注意**：写作前从设定快照中读取角色的内在驱动。场景中角色的主动行为必须可追溯到其 Urge 或 At Hand。

- **对他人认知**：（对其他角色的假设——身份/历史/气质/本质。可能正确或错误。决定了角色如何看待和预判他人的行为）
- **人生终极目标**：（最想要什么——具体、理想化、不套话。不是"变强"而是"成为能让师父说出'你出师了'的人"）
- **近期具体目标**：（数周/数月内可达成的事项。与人生目标有可追溯的路径关系）
- **今日客观日程**：（角色今天本来要做什么。在被打断/卷入事件之前的状态）
- **此刻正在做什么 (At Hand)**：（场景开始前已在进行的动作。不是"等待剧情发生"——角色总是已经在做某件事）
- **此刻的内在冲动 (Urge)**：（此刻最自私、主动的欲望——不是对叙事节拍的反应，而是在叙事节拍发生之前就存在的驱动力）
- **冲动评估 (Evaluation)**：此冲动是否与既有设定矛盾？优先级如何？角色倾向于自助还是寻求帮助？
- **具体行动 (Move)**：（为此冲动采取的物理上具体的、主动的动作。必须是读者能看到的行为，不是心理活动）

> 以下 12 种关系动态模型提炼自 `reference/novels/` 中的参考作品和通用叙事理论。

> 12 种关系模型已迁移至 [`framework/guides/reference-material.md` §三](../guides/reference-material.md#三关系模型参考12-种)。

### 使用方式

1. 选择最接近你角色关系的 1-2 个模型作为"关系骨架"
2. 在「人际关系」表中填写具体的"表层/深层态度"和"专属互动模式"时对照参考
3. 「关系变化轨迹」中的每次变化应体现模型的核心张力——模型定义了"什么会变"和"什么不会变"
4. 不要完全复制模型的表面形式——提取"为什么有效"并应用到你的具体角色身上
