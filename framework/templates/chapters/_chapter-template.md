## AI 生成模式

> 为每个场景选择 AI 生成模式。AI 根据模式自动切换行为策略。
> 详见 `framework/guides/ai-risk-mitigation.md`。

| 场景 | 模式 | 说明 |
|------|------|------|
| 1 | 标准 / 高纹理 / 声音锁定 / 回响 | |
| 2 | | |
| 3 | | |

**模式说明**：
- **标准** = 日常场景、过渡段落、普通对话。AI 正常生成 + 事后自检
- **高纹理** = 吐槽段落、情感爆发、名场面、章节结尾 200 字。AI 强制句长扰动 + 保留破句 + 过渡词砍半 + 插入非理性表达
- **声音锁定** = 多角色对话密集场景、修罗场、角色首次出场。AI 生成前调用每个角色声音卡、确保锚点词出现
- **回响** = 伏笔揭示、角色重聚、呼应前文。AI 生成前读取原始语境快照 + 插入回响锚点

## 正文

> **时空标签**：章节首尾使用标准化时空锚点，格式如下：
> `『{纪年法} {年}年 {月}月 {日}日丨周{几}丨{时}:{分}丨{地点}』`
> 纪年法/年/月/日/星期/时/分根据作品背景灵活调整，禁止模糊如"某""几"。
