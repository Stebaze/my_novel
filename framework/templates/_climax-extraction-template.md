# 爆点桥段提取模板

> 从参考作品原文中提取完整的高潮桥段序列（开端→发展→高潮→结局→余震），形式化为可复用模板。
> 每个模板必须引用原文证据——不是二手概括。

## 元信息（YAML frontmatter）

```yaml
---
pattern_id: "{author-slug}-climax-{seq}"
format_version: "1.0"
extracted_from: "{作者}·{作品}"
source_chapters: "Ch{X}-{Y}"
extracted_at: "{YYYY-MM-DD}"
spectrum: "{small|medium|large|super}"
mechanism_type: "{information_reveal|combat_decision|character_choice|multi_thread_convergence|emotional_breakthrough|power_breakthrough}"
novel_phase: "{early|mid|late}"
---
```

## 一、桥段名称

{简短、可引用的名称——2-6 个字概括此桥段的核心机制}

## 二、来源

- **作品**：{作品名}，{作者}
- **章节范围**：Ch{X}-{Y}（共 {N} 章）
- **全书阶段**：{前期/中期/后期}——占全书 {X}% 位置
- **原文摘录**（关键段落，≥2 处引用）：
  1. {高潮引爆段落，50-100 字}（Ch{X}）
  2. {余震/收束段落，50-100 字}（Ch{Y}）

## 三、阶段上下文

- **本阶段特征**：{该桥段所在的前/中/后期，此小说的叙事策略是什么？主角状态？世界观揭示程度？}
- **前序铺垫**：{此桥段之前建立了哪些压力/线索/关系？多少章的铺垫？}
- **后续影响**：{此桥段之后故事走向如何改变？}

## 四、阶段演变

> 同类型桥段在本作品其他阶段的变体。

| 阶段 | 章节 | 相同点 | 不同点 | 演变逻辑 |
|------|------|--------|--------|---------|
| 前期 | Ch{X}-{Y} | {不变的核心机制} | {规模/情感/代价的差异} | {为什么此阶段用这个变体} |
| 中期 | Ch{X}-{Y} | | | |
| 后期 | Ch{X}-{Y} | | | |

无变体则标注"此书仅在中/后期出现此类型"。

## 五、机制

此桥段如何运作？（一段话——不是描述"发生了什么"，而是描述"为什么这个结构奏效"。）

示例格式："此桥段的核心是{机制}——通过{方法}制造{效果}。读者在{阶段}感受到{情感}，在{阶段}经历{变化}。"

## 六、五阶段节拍序列

| 阶段 | 章节跨度 | 字数估 | 关键事件 | 情感曲线 | 原文证据 |
|------|---------|--------|---------|---------|---------|
| 开端 | Ch{X} | ~{N}字 | {一句话} | {进入情感}→{退出情感} | "{原文片段}"（Ch{X}） |
| 发展 | Ch{X}-{Y} | ~{N}字 | {一句话} | | |
| 高潮 | Ch{Y} | ~{N}字 | {一句话} | | |
| 结局 | Ch{Z} | ~{N}字 | {一句话} | | |
| 余震 | Ch{Z}后段/Ch{Z+1} | ~{N}字 | {一句话} | | |

> 原文证据列每阶段至少 1 条引用。

### 阶段 A：开端

- **触发条件**：{什么事件/信息启动了此桥段？}
- **初始状态**：{桥段开始时的局面——谁在哪、在做什么、面临什么}
- **进入方式**：{前章结尾如何处理→本章如何开场？（如：钩子回收式/事件中段切入式/氛围铺垫式）}
- **必须传达的信息**：{读者在开端阶段必须知道什么}

### 阶段 B：发展

- **升级机制**：{什么让事态变得更紧张/复杂？（如：新信息出现/退路封死/第三方介入）}
- **压力叠加方式**：{一次升级还是多次升级？每次升级的触发因素？}
- **可选支线/侧面描写**：{可用但非必须的次级事件——分散读者注意力或提供喘息}

### 阶段 C：高潮

- **峰值场景描述**：{高潮瞬间的具体画面——谁做了什么、什么被打破/揭示}
- **事件密度**：{约 X 件有意义事件/千字——节奏必须快于常规章}
- **叙事技法**：{此阶段采用了什么技法？（如：多线汇聚/平行剪辑/短句加速/意象堆叠）}
- **信息释放**：{高潮中揭示/回收了什么关键信息？（伏笔ID/世界观碎片/角色真相）}
- **不可逆变化**：{高潮过后什么被永久改变了？}

### 阶段 D：结局

- **解决方式**：{核心冲突如何解决？谁赢了？代价是什么？}
- **即时后果**：{高潮后立即出现的新局面——战场清理/角色反应/信息扩散}
- **状态重置 vs 状态跃迁**：{是回到日常（但日常基线变了）还是进入全新状态？}

### 阶段 E：余震

- **情感余韵**：{角色对已发生事件的情感反应——不是解决新问题，是消化旧冲击}
- **新种子**：{桥段结束后埋下的新线索/新问题/新威胁——给读者继续读的理由}
- **下一章入口**：{建议的下一章开场方式（如：日常沉降/钩子回收/时间跳跃）}
- **冷却长度**：{建议冷却 0.5-1 章——高潮强度的 1/2}
