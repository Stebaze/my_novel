# 脚本格式速查

> AVG 剧本块类型和格式规则速查。完整规范见 `framework/guides/avg-script-format.md`。

## 块类型速查

| # | 块 | 语法 | 何时用 |
|---|-----|------|--------|
| 1 | 场景头 | `【场景：地点·时间】` | 每个场景开头 |
| 2 | BGM | `【BGM：曲风/情绪】` | 场景开始 + 情绪转折 |
| 3 | 背景 | `【bg：背景ID】` | 场景开始 + 地点切换 |
| 4 | 立绘 | `【立绘：角色 表情 位置 [入场]】` | 角色登场/表情变化/退场 |
| 5 | 对话 | `角色名 [表情]：「台词」 ⏱ N.Ns` | 每句台词——表情标注必填 |
| 6 | 演出注 | `（注：动作/语气/微表情）` | 对话下方——补充视觉/听觉信息 |
| 7 | 旁白 | `【旁白】...【/旁白】` | 场景描述/内心独白/时间推移 |
| 8 | 静默/停顿 | `【bgm 静默 N秒】` / `【停顿 N秒】` | 情绪转折/蓄力/留白 |
| 9 | CG | `【CG：名称 \| 触发时机】` + 提示词 | 名场面定格——情感高潮/初遇/章末 |
| 10 | 检定 | `【检定：技能 \| 难度：NN \| 结果】` | 跑团Replay VN——关键技能判定 |
| 11 | 选择 | `【选择】` + 选项列表 | 标准AVG——玩家决策分支 |

## 立绘表情速查

| 表情 | 说明 | 典型触发 |
|------|------|---------|
| `默认` | 中性表情 | 普通对话 |
| `微笑` | 温和的笑 | 日常互动 |
| `苦笑` | 无奈的笑 | 吐槽/自嘲 |
| `开心` | 明显的喜悦 | 好事发生 |
| `生气` | 愤怒/不满 | 被惹到 |
| `害羞` | 脸红/回避视线 | 被戳中 |
| `悲伤` | 低沉/失落 | 谈及创伤 |
| `眼眶泛红` | 忍住不哭 | 苦涩瞬间 |
| `惊讶` | 瞪大眼睛 | 意外信息 |
| `无语` | 面无表情/死鱼眼 | 吐槽反应 |
| `坏笑` | 恶作剧表情 | 捉弄别人 |
| `认真` | 严肃/专注 | 重要宣言 |
| `逞强` | 假装没事但看得出 | "我没事" |

**表情变化规则**：角色情绪变了 → 更新 `【立绘：角色 新表情】` + 下一句对白 `[新表情]` 同步标注。同角色连续同表情仍需逐句标注 `[表情]`。

## 计时速查

```
秒数 = max(0.5, 中文字数 / 3.5 + 标点附加)
```

| 标点 | 附加 |
|------|:----:|
| `。！？` | +0.3s |
| `……` | +0.5s |
| `，、` | +0.15s |

- 单字回应 ≥ 1.0s（"嗯""好""不"）
- `「……」` → `⏱ —`
- 旁白默认不计时（如需配音则标 ⏱）
- 演出注不计时

## 常见错误

| 错误 | 纠正 |
|------|------|
| 角色表情变了但没更新立绘指令 | 情绪变化 = `【立绘：角色 新表情】` + 对白标注同步 |
| 对白缺少 `[表情]` 标注 | 每句对白必标表情 |
| 演出注重述对话内容 | 写看得见的——动作/表情/环境互动 |
| 演出注写抽象心理 | "她在想他"→"她把手机翻到通讯录，又翻回去" |
| bg 缺少生图提示词 | 每章章末「背景生图提示词索引」补齐 |
| CG 描述了角色样貌 | 仅写姓名——样貌从立绘规格卡获取 |
| 忘记立绘退场 | 角色离开场景 → `【立绘：角色 淡出】` |
| 忘记配音时长 | 每句对白必标 `⏱ N.Ns` |
