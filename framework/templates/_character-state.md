# 角色状态追踪

> **定位**：记录每个出场角色在每章结束时的完整状态——时间、空间、身份、身体、活动、心理、关系、物品、存活。这是后文写作时的连续性检查基准。
>
> **强制规则**：
> - 每章写作完成后，墨必须在阶段 8 将本章出场角色的终态写入本文件
> - 下一章阶段 0 必须检查前一章的状态记录完整性——缺失则阻断，要求回补
> - **不可降级、不可跳过**
>
> **读取规则**：写作 Ch{N} 前，读取本文件中"对应章节 = Ch{N-1}"的角色状态作为该角色的初始状态。

---

## Ch{N} 角色状态

> 每章按以下格式记录。复制此模板，将 `{N}` 替换为章节号，`{角色名}` 替换为实际角色名。

### 模板

```
### [{角色名}]

- **时间位置**：（故事时间线中的位置——日期/时刻/相对于关键事件的时间。例：Ch11 结束后的当天深夜）
- **空间位置**：（物理位置——建筑/区域/城市/坐标。例：不夜宫附中演练房"夜月"场景内）
- **身份/伪装**：（当前使用的身份、伪装、公开身份。例：[主角]本体 / 附身[角色C]中 / 以学生身份活动）
- **身体状态**：（受伤程度/疲惫度/中毒/异常状态/灵格状态。例：轻微疲惫，右臂肌肉拉伤未愈，灵力消耗约 60%）
- **正在进行的持续行动**：（跨章的持续活动——修炼、治疗、等待、旅行中、执行任务中等。如果本章结束了某项活动，标注"已完成"。例：正修炼元月刀法第一式，已掌握前两段）
- **心理/情绪状态**：（当前情绪基调/心理压力/心态变化。例：因目睹吕玄阳实力而略有动摇，但通过修炼重获信心）
- **持有重要物品**：（本章获得/携带的重要物品、武器、道具。标注获得章节。例：古镜碎片(Ch1)、元月刀(Ch11 兑换)）
- **关键关系变化**：（本章中与其他角色关系的推进/转折/恶化——与前一章相比。例：与[角色C]互信加深；[角色D]主动提出陪同训练）
- **存活状态**：🟢 存活 / 🔴 Ch{X} 已死亡（仅回忆/幻觉） / 🟡 Ch{X} 失踪/状态未知 / ⚪ 未出场

**章末位置锚点**（用于下一章开场定向——墨在写下一章开局时必须参照此字段）：
- （一句话——角色在本章最后一个场景结束时的精确位置和状态。例：[主角]躺在床上，闭上眼——准备通过古镜碎片再次穿越附身。）
```

### 填写规范

1. **必填维度**：以上 9 个维度 + 章末位置锚点，全部必填。无法确定时填"待确认"并标注原因。
2. **精确定向**："章末位置锚点"是下一章墨开场时角色位置的强制约束——墨必须从这个精确位置和状态开始写。
3. **跨章持续行动**：如果角色在做的某件事跨越多章（如"三日修炼""长途旅行""养伤"），在"正在进行的持续行动"中持续标注进度，直到标注"已完成"。
4. **身体状态连续性**：受伤/中毒/疲惫等状态必须在后续章中体现影响——不能上一章记录"右臂受伤"下一章角色正常使用右臂而不加说明。
5. **存活状态**：一旦标记为 🔴 已死亡，该角色不可在"当前时间线"中再出现——只可在回忆/幻觉/他人冒充中出现且必须在文本中明确标记。
