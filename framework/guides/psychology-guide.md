## 一、角色心理结构模型

### 1.1 三重人格动力系统（弗洛伊德）

每个角色的行为由三层人格的互动驱动。这不是标签，而是**冲突引擎**。

| 层次 | 原则 | 内容 | 叙事功能 |
|------|------|------|----------|
| **本我 (Id)** | 快乐原则 | 原始欲望——复仇、占有、毁灭、生存、情欲 | 角色的"想要"——推动行动的原始燃料 |
| **自我 (Ego)** | 现实原则 | 理性调解——分析利弊、抑制冲动、寻找折中 | 角色的"能做"——在社会规则下的实际选择 |
| **超我 (Superego)** | 道德原则 | 良心审判——道德准则、内化的社会规范、理想自我 | 角色的"应该"——自我谴责与道德约束的来源 |

**Agent 使用方式**：

1. **角色创建时**：为每个主要角色填写三重人格剖面
2. **写角色决策场景时**：明确当前决策是哪个"我"在主导
3. **审阅时**：检查角色重大决策是否与其三重人格结构一致

**三重人格剖面模板**：

```
角色：[角色名]
- 本我核心欲望：（一句话——ta 最原始、最不理性的想要什么）
- 自我调节策略：（ta 如何在社会规则下满足/压抑本我）
- 超我道德锚点：（ta 内心不可逾越的那条线是什么）
- 三重冲突爆发条件：（什么情况下三层人格会剧烈冲突）
- 通常哪层胜出：（ta 在压力下的默认选择）
```

**叙事应用**——三幕式与三重人格的对应：

| 幕 | 主导人格层 | 叙事状态 |
|----|-----------|----------|
| 第一幕 | 超我主导 | 角色生活在道德/社会规范中，本我被压抑 |
| 第二幕 | 本我释放 | 触发事件使本我欲望突破束缚，角色行为转变 |
| 第三幕 | 自我重构 | 本我欲望带来后果与危机，角色重构心理平衡 |

**实例**（《哈姆雷特》）：
- 本我：渴望为父复仇的原始愤怒
- 自我：设计"捕鼠器"戏文来验证克劳狄斯的罪行（理性策略）
- 超我：对弑叔行为的道德内疚 → 延宕
- 冲突结果：三重人格的拉锯导致了悲剧

### 1.2 防御机制：角色面对压力时的行为密码

防御机制是角色面对内在/外在威胁时**无意识采用的心理策略**。它让看似矛盾的行为变得可理解，是塑造"表里不一"角色的核心工具。

| 防御机制 | 心理逻辑 | 行为表现 | 适合的角色类型 |
|----------|----------|----------|--------------|
| **压抑** | 将不可接受的欲望/记忆推入无意识 | 记忆空白、突然的情绪冻结、对特定话题的回避 | 有创伤过去的角色 |
| **投射** | 将自身缺点/欲望归咎于他人 | 指责别人有自己实际上也有的问题 |  hypocrite 型反派/配角 |
| **反向形成** | 将真实情感转化为相反的表现 | 越喜欢越冷漠/越在意越挑剔 | 傲娇/冷娇型角色 |
| **退行** | 在压力下退回早期发展阶段的应对方式 | 成熟角色突然幼稚、暴怒、哭闹 | 平时强势但在特定人面前软弱的角色 |
| **升华** | 将本能欲望转化为社会认可的活动 | 将攻击欲转化为竞技精神，将情欲转化为艺术创作 | 艺术家/竞技者型角色 |
| **合理化** | 用看似合理的理由解释非理性行为 | "我不是不敢，是没必要" | 嘴硬型角色 |
| **转移** | 将情绪从真正目标转移到替代目标 | 被老板骂 → 回家对家人发火 | 权力不对等场景中的角色 |
| **否认** | 拒绝接受现实 | "不可能，她不会死的" | 面对重大丧失的角色 |

**Agent 使用方式**：

1. 为主要角色指定 1-2 种主导防御机制
2. 在角色面对压力/威胁/尴尬时，让行为先通过防御机制的"滤镜"再输出
3. 防御机制不是角色自己选择的——它是无意识的，所以角色自己不会说"我在反向形成"

**防御机制触发检查**（写角色压力场景时自查）：
- 这个角色面对压力时，ta 的无意识第一反应是什么？
- ta 自己知道自己在用这个防御机制吗？（通常不知道）
- 这个防御机制有没有给 ta 带来新麻烦？（好叙事——防御机制本身成为问题的一部分）

### 1.3 荣格原型系统

原型是超越个体经验的、存在于集体无意识中的原始意象。为角色赋予原型，使其行为具有**文化普遍性和象征深度**。

#### 核心原型

| 原型 | 核心驱动力 | 叙事功能 | 典型角色 |
|------|-----------|----------|----------|
| **英雄** | 证明自我价值 + 保护他人 | 主角的默认原型 | 哈利·波特、辛巴 |
| **阴影** | 被压抑的黑暗面寻求表达 | 反派/主角的黑暗面 | 伏地魔（哈利的阴影）、六耳猕猴（孙悟空的阴影） |
| **阿尼玛/阿尼姆斯** | 男性内心的女性意象 / 女性内心的男性意象 | 爱情线的心理深度 | 《七月与安生》双女主 |
| **大母神** | 滋养/保护/吞噬 | 导师/庇护者/控制型母亲 | 索尼娅（《罪与罚》） |
| **智慧老人** | 传授智慧 + 引导成长 | 导师角色 | 邓布利多、拉飞奇 |
| **骗子/捣蛋鬼** | 打破规则 + 制造混乱 | 喜剧调剂/情节催化剂 | 洛基、韦小宝 |
| **人格面具** | 社会身份与角色扮演 | 角色的"公开版本" | — |
| **自性** | 整合所有对立面后的完整人格 | 角色成长的终极目标 | — |

#### 人格面具 vs 阴影：社会身份与真实自我的张力

这是塑造角色"表里不一"的最强工具：

```
人格面具 (Persona) = 角色在公众面前展现的社会身份
阴影 (Shadow)     = 角色内心被压抑、否认或忽视的特质

叙事张力 = 人格面具与阴影之间的裂隙
```

**Agent 使用步骤**：

1. **设计人格面具**：确定角色在公众面前展现的社会身份与行为模式
2. **挖掘阴影内容**：识别角色内心被压抑的欲望/恐惧/黑暗面
3. **安排面具崩塌**：设计触发事件使角色无法继续维持人格面具
4. **构建阴影整合**：规划角色如何通过危机经历整合阴影，实现人格成长

**阴影整合叙事节奏**：
```
压抑期（面具牢固）→ 裂隙期（阴影泄露）→ 崩塌期（面具碎裂）→ 整合期（承认阴影，人格完整）
```

### 1.4 理论融合：角色心理结构完整剖面

将弗洛伊德与荣格融合，形成角色的完整心理剖面：

```
角色心理结构完整剖面：

1. 核心原型：（英雄/阴影/阿尼玛/大母神/智慧老人/捣蛋鬼）
2. 三重人格：
   - 本我欲望：
   - 自我策略：
   - 超我锚点：
3. 主导防御机制：（1-2 种）
4. 人格面具：（公开形象）
5. 阴影内容：（被压抑的特质）
6. 成长弧线：（从压抑/分裂 → 整合/完整）
7. 关键触发点：（什么事件打破心理平衡）
8. 象征系统：（与角色心理关联的意象——镜子/水/阴影/笼子/迷宫/光）
```

### 1.5 犯罪心理结构模型：反派独立评估框架

> **区别于弗洛伊德三重人格**：弗洛伊德三层次（本我/自我/超我）是**冲突引擎**——解释角色内部的动力拉锯。犯罪心理学三层次（认知/情感/动机）是**模式识别工具**——解释反派如何形成并维持异常行为模式。两者互补，不可替代。

**三层次定义**：

| 层次 | 核心问题 | 关键维度 | 叙事功能 |
|------|----------|----------|----------|
| **认知层 (Cognitive)** | 反派如何看待世界？ | 道德认知偏差 / 他人工具化程度 / 现实检验能力 | 决定行为策略——如何规划、合理化、看待后果 |
| **情感层 (Affective)** | 反派如何感受？ | 共情能力谱系 / 情感幅度 / 快感来源 | 决定与读者的情感距离——完全无法共情的反派制造"寒意" |
| **动机层 (Motivational)** | 反派为什么行动？ | 驱动类型 / 目的性(手段vs目的) / 内在一致性 | 决定行动的可预测性和冲突升级路径 |

**三层次互动关系**：

```
认知层（"世界是可操纵的游戏"）
    ↓ 塑造
情感层（"他人的痛苦与我无关"）
    ↓ 驱动
动机层（"我需要更多的控制感"）
    ↓ 外化
犯罪行为（每次成功犯罪强化三层次结构）
    ↓ 反馈回认知层（"看，我一直都是对的"）
```

**Agent 使用方式**：

1. **创建反派时**：先填犯罪心理三层次，再填 §1.1-§1.4 的弗洛伊德/荣格剖面
2. **对于纯粹恶意反派**：重点在认知层和情感层——动机层反而是最简单的（"他这么做是因为他享受"）
3. **对于可救赎反派**：三个层次都需详细设计，特别是认知层中的"潜在转变触发点"
4. **审阅时**：检查反派行为是否与其三层次结构一致

**犯罪心理三层次评估模板**：

```
反派：[角色名]
恶意类型定位：[纯粹恶意 / 偏离正常行为 / 混合型]
- 认知层 — 道德认知：(社会规则是工具 / 道德是弱者的借口 / 其他)
         他人定位：(可利用的资源 / 有独立价值的个体 / 混合)
         现实检验：(认知清晰但价值扭曲 / 存在妄想或幻觉)
- 情感层 — 共情能力：(完全缺失 / 选择性共情 / 正常但扭曲)
         情感幅度：(情感淡漠 / 正常幅度 / 情绪极端波动)
         快感来源：(正常来源 / 支配控制 / 他人痛苦)
- 动机层 — 驱动类型：(权力 / 快感 / 报复 / 强迫 / 意识形态 / 交易 / 虚无)
         目的性：(犯罪是手段 / 犯罪是目的 / 混合)
         内在一致性：(动机→行为是否自洽？矛盾点在哪？)
```

### 1.6 反派心理类型学：24 种模板速查

> **定位**：与 §4.2「阴影反派设计」（4 种阴影类型：镜像/压抑/集体/诱惑）互补——阴影类型解决"反派与主角的关系"，本类型学解决"反派自身的心理结构"。24 种模板完整详情见 `framework/guides/villain-design-guide.md`。

**恶的来源四象限**（使用前先定位）：

| 来源 | 类别 | 核心命题 | 数量 |
|------|------|----------|:----:|
| **病理性的** | A / B / E 类 | "恶来自心理结构/障碍" | 11 |
| **选择性的** | C 类 | "ta 没有病。ta 做了选择。" | 6 |
| **关系性的** | D 类 | "恶不在个体心理中——在互动模式里" | 4 |
| **观念性的** | F 类 | "恶的驱动力是观念，不是心理" | 2 |

#### A 类：纯粹恶意模板（不可调和，5 种）

| # | 模板 | 别名 | 核心 | 一句识别 |
|---|------|------|------|----------|
| A1 | 反社会人格障碍型 | 「优雅的恶魔」 | 魅力伪装 + 他人工具化 + 无内疚 | "道德是工具，他人是资源" |
| A2 | 精神病态型 | 「无心的猎人」 | 情感真空 + 纯粹理性 + 无恐惧/爱/恨 | "不理解'道德'——像无法感知颜色的盲人" |
| A3 | 施虐型 | 「痛苦的艺术家」 | 痛苦→快感 + 仪式化犯罪 + 控制欲 | "你的恐惧是他的氧气" |
| A4 | 集体犯罪型 | 「群体的傀儡师」 | 意识形态操控 + 去个性化 + 不亲自动手 | "他制造系统，让系统替他作恶" |
| A5 | 模仿/崇拜型 | 「偶像的追随者」 | 理想化前人犯罪 + 认知失调 + 为被看见 | "犯罪不是目的——是想成为像偶像那样的人" |

#### B 类：可救赎模板（转变需付巨大代价，3 种）

| # | 模板 | 别名 | 核心 | 救赎可能 |
|---|------|------|------|:--------:|
| B1 | 边缘型人格障碍型 | 「情绪的囚徒」 | 情绪极端波动 + 对被抛弃敏感 + 冲动犯罪 | ⭐⭐⭐ |
| B2 | 强迫型人格障碍型 | 「仪式的囚徒」 | 焦虑驱动 + 仪式化犯罪 + 强烈道德冲突 | ⭐⭐ |
| B3 | 自恋型人格障碍型 | 「破碎的镜子」 | 自我夸大 + 自尊受损→攻击 + 需要崇拜 | ⭐ |

#### C 类：非病理型恶（恶来自选择，6 种）

| # | 模板 | 别名 | 核心 | 关键区分 |
|---|------|------|------|----------|
| C1 | 虚无主义者 | 「意义毁灭者」 | 论证"一切无意义" + 逻辑武器 | "他可能是对的——无法被证明是错的" |
| C2 | 纯粹交易者 | 「恶是生意」 | 一切有价格 + 专业主义 + 不恨不爱 | "这不针对个人——是严格意义上的生意" |
| C3 | 腐化理想主义者 | 「善的滑坡」 | 出于善→方法代替目的→无法回头 | "他就是主角在某个节点做了不同选择的版本" |
| C4 | 捕食者 | 「动物逻辑」 | 看到弱点→利用 + 前道德 + 不解释 | "没有心理深度可挖掘。空洞本身就是答案" |
| C5 | 审判者 | 「自以为义的执行者」 | 逻辑自洽+真诚正义感+第一前提错 | "他说的 90% 是对的——你怎么反对？" |
| C6 | 懦弱共谋者 | 「不作为的恶」 | 知道发生什么+不想惹麻烦+累积 | "恶的滋生地不是恨，是'我不想惹麻烦'" |

#### D 类：关系型恶（恶在互动模式中，4 种）

| # | 模板 | 别名 | 核心 | 关键区分 |
|---|------|------|------|----------|
| D1 | 煤气灯操控者 | 「现实扭曲者」 | 系统性扭曲对方现实感知 + 共情武器化 | "受害者无法求救——不确定自己是否受伤害" |
| D2 | 寄生者 | 「以他人为食」 | 靠他人资源/善意为生 + 用完即弃 | "不享受痛苦——享受不劳而获" |
| D3 | 腐蚀者 | 「以堕落为乐」 | 快感在于让好人做坏选择 | "不对主角感兴趣——对主角身边最好的人感兴趣" |
| D4 | 背叛者 | 「亲密是武器」 | 先建立信任亲密 + 再利用信任 | "不是'从来不是朋友'——是'是朋友但还是背叛'" |

#### E 类：补充病理型（DSM 遗漏，3 种）

| # | 模板 | 别名 | 核心 | 关键区分 |
|---|------|------|------|----------|
| E1 | 偏执型 | 「恐惧驱动的先发制人」 | 被害信念 + 恐惧驱动 + "自卫"式攻击 | "不是恨你——是'确定'你会害他" |
| E2 | 妄想型 | 「使命驱动的清晰疯狂」 | 固定错误信念 + 妄想内完全理性 | "比你还冷静，但他的现实不是你的现实" |
| E3 | 表演型 | 「注意力是氧气」 | 自我价值=被看见 + 犯罪是表演 | "不被看见=不存在。越被忽视越极端" |

#### F 类：哲学/审美型恶（恶来自观念，2 种）

| # | 模板 | 别名 | 核心 | 关键区分 |
|---|------|------|------|----------|
| F1 | 无聊之神 | 「形而上消遣」 | 无限力量+时间→无聊 + 消遣驱动 | "什么都见过。你的一生是半集电视" |
| F2 | 审美毁灭者 | 「毁灭即艺术」 | 毁灭是作品 + 追求毁灭的"美感" | "讲究的不是更多血——是更好的构图" |

**恶的来源诊断决策树**（Agent 定位反派类型时使用）：

```
Q1: 恶的来源是病理性的吗？（人格障碍/心理疾病）
    是 → Q1a: 是反社会/精神病态/施虐/集体操控/模仿崇拜？ → A 类（不可调和）
         Q1b: 有转变可能（BPD/OCD/NPD）？ → B 类（可救赎）
         Q1c: 是偏执/妄想/表演驱动？ → E 类（补充病理）
    否 → Q2

Q2: 恶的来源是关系性的吗？（恶在互动模式中而非个体心理内）
    是 → D 类（煤气灯/寄生/腐蚀/背叛）
    否 → Q3

Q3: 恶的来源是观念/哲学性的吗？
    是 → F 类（无聊之神/审美毁灭者）
    否 → Q4

Q4: 恶是 ta 的选择吗？（不是病、不是关系、不是观念——就是选择了作恶）
    是 → C 类（虚无/交易/腐化理想/捕食/审判/懦弱共谋）

重叠：一个角色可同时属于多个类别。
  例：C3(腐化理想) + C5(审判者) = 为正义滑入恶
  例：D4(背叛者) + E1(偏执型) = 偏执→不信任→背叛
  例：F2(审美毁灭) + A3(施虐型) = 享受痛苦+追求毁灭美学
```

### 1.7 角色多样性工程：七维设计检验

> **定位**：系统性确保角色群在设计层面具有多维差异。创建/审阅角色群时使用。

**七个设计维度**：

| 维度 | 谱系 | 检验 |
|------|------|------|
| **道德轴心** | 德性伦理 → 义务伦理 → 后果伦理 → 关系伦理 → 无伦理 → 反伦理 | 至少 3 种不同 |
| **认知风格** | 分析型 → 直觉型 → 经验型 → 系统型 → 叙事型 | 主要角色不能全部同一风格 |
| **情感调节** | 外化型 → 内化型 → 理智化型 → 躯体化型 → 分裂型 | 同一压力下不同角色不同反应 |
| **欲望结构** | 拥有 / 成为 / 证明 / 逃避 / 消除 / 感受 | 角色群至少覆盖 3 种 |
| **依恋模式** | 安全型 / 焦虑型 / 回避型 / 混乱型 | 不能全部相同 |
| **成长轴线** | 成长型 / 硬化型(坚守对的) / 僵化型(坚守错的) / 崩溃型 / 无变化型 | 不是每个角色都需要成长弧线 |
| **秘密/透明** | 有秘密且隐藏 / 被察觉 / 被看穿 / 无秘密被误认有 / 完全透明 | 利用秘密不对称制造认知错位 |

**多样性检验清单**：

- [ ] 有没有两个角色的道德轴心完全相同？
- [ ] 有没有两个角色的认知风格完全相同？
- [ ] 同一压力事件下不同角色是否有不同情感反应？
- [ ] 角色群的欲望类型是否覆盖至少 3 种？
- [ ] 有没有角色的存在只为"推动情节"（功能性角色过多）？
- [ ] 纯粹恶意反派至少 1 个？（如果故事类型需要）
- [ ] 有没有"太过相似"的两个角色可被合并？

## 二、情感工程系统

### 2.1 VAD 三维情感模型

VAD（Valence-Arousal-Dominance）将情感量化为三个维度，为 AI 生成情感化文本提供精确的参数控制。

| 维度 | 范围 | 含义 | 叙事对应 |
|------|------|------|----------|
| **Valence（效价）** | -1.0（负面）→ +1.0（正面） | 情绪的正负面 | 场景的情感基调 |
| **Arousal（唤醒度）** | 0（平静）→ 1.0（激动） | 情绪的强度 | 场景的张力水平 |
| **Dominance（优势度）** | 0（无力）→ 1.0（掌控） | 对情境的控制感 | 角色的权力位置 |

**常见情感状态的 VAD 参数**：

| 情感状态 | Valence | Arousal | Dominance | 适用场景 |
|----------|---------|---------|-----------|----------|
| 平静满足 | +0.6 | 0.2 | 0.7 | 日常温馨场景 |
| 兴奋激动 | +0.8 | 0.8 | 0.6 | 爽点/高潮 |
| 焦虑不安 | -0.5 | 0.6 | 0.2 | 悬念/压力场景 |
| 愤怒 | -0.6 | 0.8 | 0.5 | 冲突爆发 |
| 悲伤 | -0.8 | 0.3 | 0.1 | 情感低谷/丧失 |
| 恐惧 | -0.7 | 0.7 | 0.1 | 恐怖/紧张 |
| 释然 | +0.4 | 0.3 | 0.5 | 危机化解后 |
| 怀旧/感伤 | -0.3 | 0.3 | 0.3 | 回忆/告别 |
| 坚定/燃 | +0.7 | 0.7 | 0.8 | 决心/反杀前 |
| 羞耻/尴尬 | -0.4 | 0.5 | 0.2 | 社死/被揭穿 |

**Agent 使用方式**：

1. **写场景前**：设定场景的 VAD 目标参数
2. **写情感描写时**：用 VAD 参数引导 AI 选择描写策略——高效价 + 高唤醒 ≠ 感动（差别在 Dominance）
3. **审阅时**：检查场景实际交付的情感是否匹配目标 VAD

**VAD 引导的情感描写策略**：

```
如果目标是 Valence=-0.8, Arousal=0.3（悲伤）：
  ✅ "他低头看着鞋尖，雨水顺着发梢滴进衣领，凉得像是谁的手指轻轻碰了他一下。"
  ❌ "他感到心里一阵难过。"（说教式，无感官）

如果目标是 Valence=+0.7, Arousal=0.7（燃）：
  ✅ "我盯着那个数字。3.7秒。够了。"
  ❌ "他心中充满了勇气和决心。"（空洞的情感形容词）
```

### 2.2 情感弧线设计

每章/每卷应有明确的情感弧线——不是平的，是有起伏的。

**章节级情感弧线模板**：

```
起始情感 → 发展/转折 → 峰值情感 → 回落/余韵

例：平静(-0.2,0.3) → 紧张(-0.4,0.6) → 燃(+0.7,0.8) → 释然(+0.4,0.3)
```

**卷级情感弧线模板（重生/逆袭文）**：

```
低谷（重生前悔恨）→ 上升（第一次逆袭）→ 小高潮（赚第一桶金）
  → 低谷（被对手打压）→ 大高潮（打脸对手）→ 余韵（确立新目标）
```

**情感弧线自检**：
- 有没有连续 3 章情感强度不变？（→ 读者审美疲劳）
- 有没有连续 2 章 Valence 完全一致？（→ 需要插入情感转折）
- 高潮后的回落是否足够？（→ 持续高潮 = 没有高潮）

### 2.3 心流理论：读者沉浸度控制

心流 = 挑战与技能的动态平衡。在叙事中表现为：**读者感受到的挑战难度与角色能力之间的匹配**。

**心流参数**：

| 参数 | 含义 | 范围 |
|------|------|------|
| 冲突强度 | 本章核心冲突的级别 | 1（日常对话）→ 5（生死抉择） |
| 能力匹配度 | 角色当前能力与挑战的匹配程度 | 低（碾压）/ 中（势均力敌）/ 高（以弱战强） |
| 节奏比例 | 铺垫/冲突/收尾的时间分配 | 如 30%-50%-20% |
| 伏笔密度 | 本章埋设/揭示的伏笔数量 | 1-3 个/章 |

**心流断裂检测**（审阅时检查）：
- 冲突强度连续 3 章不变 → 读者进入"自动播放"模式
- 挑战远超角色能力且无合理化解 → 读者感到"开挂"不适
- 挑战远低于角色能力且无其他看点 → 读者感到"无聊"
- 节奏比例失衡（如 80% 铺垫 10% 冲突）→ 读者跳章

**章节心流参数模板**：

```
第 X 章心流参数：
- 冲突强度：__ / 5
- 能力匹配：低 / 中 / 高（角色需要____才能应对）
- 节奏：__% 铺垫 - __% 冲突 - __% 收尾
- 心流锚点：（角色用何种具体能力/策略应对挑战）
- 伏笔密度：埋 __ 处 / 揭示 __ 处
```

### 2.4 心流与视觉：图文双重沉浸

> 插画是心流体验的放大器。论文数据显示，图文结合使读者心流体验从 51% 提升至 73%，认知参与度提升 35%，情感投入度提升 42%。
>
> 插画通过三层机制增强心流：**多感官锚定**（视觉细节触发感官记忆）→ **叙事锚点**（具象参照降低认知负荷）→ **情感锚定**（色彩/光影直接作用于情绪）。
>
> 关于插画如何增强心流、四种图文互动模式的选用策略、多平台 prompt 生成，详见 `framework/guides/illustration-guide.md`。

## 三、感官锚定系统（普鲁斯特效应）

### 3.1 原理

嗅觉信息通过嗅球直接传递到杏仁核（情绪处理）和海马体（记忆中枢），绕过丘脑过滤 → 产生更直接、更强烈的情感记忆。在写作中，精准的感官描写比抽象的心理陈述更有情感感染力。

### 3.2 感官-情感映射库

| 感官类型 | 感官锚点示例 | 常见情感映射 | VAD 参数范围 | 适合场景 |
|----------|-------------|-------------|-------------|----------|
| **嗅觉** | 雨后泥土味 | 悲伤、怀旧、孤独 | V=-0.6~-0.3, A=0.2~0.4 | 告别/回忆/独处 |
| **嗅觉** | 旧书纸味 | 怀旧、温暖、安全感 | V=+0.3~+0.5, A=0.2~0.3 | 发现秘密/回到老地方 |
| **嗅觉** | 消毒水味 | 焦虑、恐惧、脆弱 | V=-0.5~-0.3, A=0.4~0.6 | 医院/受伤场景 |
| **听觉** | 怀表滴答声 | 焦虑、期待、时间紧迫 | V=-0.4~+0.4, A=0.5~0.7 | 倒计时/等待 |
| **听觉** | 远处火车汽笛 | 孤独、离别、向往远方 | V=-0.5~-0.2, A=0.3~0.5 | 告别/思念 |
| **视觉** | 潮湿的落叶 | 孤独、回忆、失落、时间流逝 | V=-0.7~-0.5, A=0.3~0.5 | 秋季/转折/角色独处 |
| **视觉** | 黄昏的长影子 | 不舍、过渡、不安 | V=-0.4~+0.1, A=0.3~0.5 | 日暮/临界时刻 |
| **触觉** | 冰凉的金属扶手 | 孤独、疏离、危险 | V=-0.6~-0.2, A=0.3~0.6 | 进入陌生/危险环境 |
| **触觉** | 粗糙的毛毯裹紧 | 安全感、被保护、脆弱中的温暖 | V=+0.3~+0.6, A=0.2~0.3 | 受伤后被照顾 |
| **味觉** | 过于甜腻的咖啡 | 伪装、强撑、苦涩中的自我欺骗 | V=-0.2~+0.3, A=0.2~0.4 | 日常中的情绪暗流 |

### 3.3 感官锚定提示词模板

```
通过 [感官类型] 描写触发 [情感目标]，要求：
1. 感官锚点：[具体感官细节]
2. 情感映射：[情感描述]
3. 多感官协同：[补充至少一种其他感官]
4. VAD 目标：valence=___, arousal=___
5. 避免：直接用情感形容词（"他感到悲伤"→用感官细节替代）
```

**实例**：

```
❌ "他走进老房子，心里涌起一股怀旧的情绪。"
✅ "他推开木门，灰尘在正午的光柱里翻飞。老樟木箱的味道——和二十年前一模一样。
    他站了一会儿，手指抚过门框上一道浅痕。那是他七岁时用铅笔刀划的。"
    → 嗅觉（樟木）+ 视觉（光柱/灰尘）+ 触觉（门框划痕）= 怀旧
```

### 3.4 寒意氛围工程：纯粹恶意场景的感官设计

> **定位**：现有感官-情感映射库（§3.2）覆盖了怀旧/焦虑/悲伤/温暖等常见情感，但缺少纯粹恶意场景所需的特定感官技巧。本子章节填补这个空白。
>
> **核心挑战**：不是"让读者害怕"（那是恐怖小说的做法），而是"让读者感到一种安静的不安"——意识到对面的这个人，和自己是不同的物种。

**五大寒意锚定技法**：

| 技法 | 机制 | 效果 | 实例 |
|------|------|------|------|
| **正常中的偏移** (Uncanny Valley) | 场景 99% 正常——但 1% 的细节"不对" | 读者不确定"是不是自己多心了"——这正是不安的来源 | 反派杀人后洗手，水龙头开到最大，洗了整整三分钟——不是慌张的洗，是像在洗一只杯子 |
| **移情的屏障** | 读者试图共情反派——遇到无法穿透的透明墙 | 不是"冷酷"——冷酷暗示有情感在压制。是"平静"——暗示没有东西需要压制 | 反派看着受害者的眼睛，表情是完全的专注——像一个昆虫学家在观察一只稀有蝴蝶 |
| **物的视角** | 用对待物的方式对待人——不是虐待，是"无视其为人" | 比刻意的侮辱恐怖一百倍——因为不经意的去人化暗示了认知层面的根本差异 | 反派在对话中自然地用"它"指代某个受害者——不是故意侮辱，是真的没往"这是个人"的方向想 |
| **反高潮的平静** | 在暴力/恐怖峰值后，立刻切到极其日常、极其平静的细节 | 让读者看到——对这个人来说，恐怖和午饭是同一级别的事件 | 一场残忍审讯结束后——"他看了眼手表。刚好是午饭时间。食堂今天有番茄炒蛋。" |
| **缺席的罪责** | 反派完全不理解"内疚"这个概念——不是否认，是概念层面不存在 | "辩解"的潜台词是"我知道你可能认为我错了"。缺席的罪责没有辩解——只有困惑 | 主角："你杀了那么多人，怎么睡得着？"反派（真诚困惑）："这和睡觉有什么关系？" |

**寒意分级系统**：

| 级别 | 读者感受 | 反派特质 | VAD 参数 (V/A/D) | 使用频率建议 |
|:----:|----------|----------|:------------------:|------------|
| L1 不安 | "这个人有点奇怪" | 轻微的社会规范偏离 | -0.3 / 0.4 / 0.3 | 自由使用 |
| L2 不适 | "我不确定是否想和ta独处" | 共情的明显缺失 | -0.5 / 0.2 / 0.8 | 每 2-3 章 |
| L3 恐惧 | "这个人非常危险且不可预测" | 暴力 + 不可预测性 | -0.6 / 0.5 / 0.4 | 每 5 章 |
| L4 认知断裂 | "我和ta不是同一个物种" | 根本性的认知/情感差异 | -0.7 / 0.3 / 0.6 | 每 10 章 |
| L5 形而上恐惧 | "恶是真实存在的" | 纯粹恶意 + 完全自洽的逻辑 | -0.8 / 0.2 / 0.7 | 每卷 1-2 次 |

**Agent 使用方式**：

1. **写反派场景前**：确定目标寒意级别（L1-L5）
2. **写反派行为时**：从五大技法中选择至少 1 种执行
3. **写反派对话时**：检查是否有移情屏障或罪责缺席——反派是否在"辩解"？如果是纯粹恶意反派，改为"困惑"
4. **审阅时**：比较实际交付的寒意与目标级别——是否有段落达到了目标？

**寒意场景自检清单**（写完后自查）：

- [ ] 反派的"恐怖"是通过行为/细节展示的，还是通过形容词告知的？
- [ ] 有没有至少一处"正常中的偏移"或"反高潮的平静"？
- [ ] 反派有没有进行"辩解"？（纯粹恶意反派 → 辩解 = 失败）
- [ ] 场景交付的寒意级别是否达到了目标？（≤L2 不需要特别克制；≥L3 必须有日常场景缓冲）
- [ ] 连续使用寒意场景是否超过 2 章？（超过 → 读者麻木）

> **跨模板交叉使用**：当反派同时匹配多个模板时，寒意技法的选择和组合需要额外协调（如 A2+F2 交叉型中，A2 的"情感真空"与 F2 的"审美满足"存在结构性张力）。详细指引见 `framework/guides/villain-design-guide.md` §5.4「跨模板交叉组合指南」。

## 四、剧情规划的心理引擎

### 4.1 英雄之旅映射表

### 4.2 阴影原型与反派设计

反派不应只是"坏人"——ta 应该是主角**内心黑暗面的外化**。

**阴影反派设计步骤**：

1. **确定主角的阴影内容**：主角内心被压抑的特质是什么？
2. **设计阴影实体**：创造一个角色/势力，其核心特质 = 主角阴影的具象化
3. **构建镜像关系**：反派与主角在某关键维度上惊人地相似（出身/经历/欲望/选择）——区别在于做出了不同选择
4. **规划阴影对抗**：每次对抗不只是力量对决，更是主角面对内心黑暗面的时刻
5. **设计阴影整合**：主角最终如何处理与阴影的关系？消灭/和解/承认共存？

**阴影类型**：

| 阴影类型 | 核心特质 | 主角的对应面 | 实例 |
|----------|----------|-------------|------|
| 镜像阴影 | 与主角相似但选择了不同的路 | "如果我在那个时刻也做了那个选择" | 哈利 vs 伏地魔 |
| 压抑阴影 | 主角否认但暗中拥有的欲望 | 主角不愿承认的自我 | 杰基尔 vs 海德 |
| 集体阴影 | 社会/组织压抑的集体黑暗面 | 主角作为系统中的个体 | "小小人"（1Q84） |
| 诱惑阴影 | 以主角想要的东西做交易 | 主角的贪欲/恐惧/软弱 | 恶魔契约类 |

### 4.3 心理冲突驱动情节法

让情节发展不是"作者安排"而是"角色的心理结构必然导致"。

**方法**：
1. 确定主角的核心心理冲突（本我 vs 超我 / 人格面具 vs 阴影）
2. 安排触发事件——打破心理平衡
3. 让角色的防御机制产生"副作用"——形成新的困境
4. 困境升级 → 旧防御机制失效 → 角色必须改变或崩溃
5. 改变/崩溃 → 心理结构重组 → 角色成长

**实例**（罪与罚型情节）：
```
核心心理冲突：主角的"超人理论"（本我：权力欲）vs 道德感（超我）
触发事件：经济困境 + 对高利贷老太的仇恨
防御机制：合理化（"杀一个对社会无用的人不是罪"）
副作用：犯罪后的内疚与 paranoid
困境升级：警察追查 + 内心崩溃
重组：在索尼娅引导下忏悔 → 阴影整合
```

### 4.4 不可调和冲突设计：让反派无法被"理解"化解

> **定位**：与 §4.3「心理冲突驱动情节法」（处理主角内部冲突）互补。本子章节处理**主角与反派之间的结构性矛盾**——为什么有些冲突不能通过"理解""谈判""感化"来解决。

**六种结构性冲突类型**：

| 类型 | 机制 | 不可调和点 | 唯一解决方案 | 典型反派匹配 |
|------|------|-----------|-------------|-------------|
| **价值观存在论** | 不是"你对还是我对"——是"你我在同一世界只能有一个" | 存在本身就互相否定 | 一方的存在被终结 | C1 虚无主义者、C5 审判者、A4 集体犯罪型 |
| **认知不可通约** | 双方使用完全不同的"逻辑操作系统"——无法翻译、无法沟通 | 沟通在概念层面就不可能 | 一方的认知框架被摧毁 | A2 精神病态、F1 无聊之神、E2 妄想型 |
| **快感依赖** | 反派的快感/满足直接依赖于主角方的痛苦/屈服 | 和平共处让反派失去存在意义 | 剥夺快感来源或终结 | A3 施虐型、D3 腐蚀者 |
| **身份本体** | 反派的自我认同依赖于"与主角方对抗"这个行为本身 | 和解 = 身份死亡 | 暴力终结（身份不需要投降） | A5 模仿型、B3 自恋型、E3 表演型 |
| **意义否定** | 反派否定主角一切行为的意义前提 | 无法辩论——ta否定的是辩论的框架本身 | 主角的"信仰一跃"（不选择认同） | C1 虚无主义者、F1 无聊之神 |
| **结构性/系统** | 反派不是一个人，是一个系统/制度/意识形态 | 对方不是一个可以"说服"或"感化"的个体 | 系统内部的自我瓦解 | A4 集体犯罪型、C6 懦弱共谋者 |

**冲突不可调和度评估矩阵**：

| 冲突类型 | 可通过理解解决？ | 可通过谈判解决？ | 可通过武力解决？ | 唯一解决方案 |
|----------|:---:|:---:|:---:|------------|
| 价值观存在论 | ✗ | ✗ | △ | 一方的存在被终结 |
| 认知不可通约 | ✗ | ✗ | △ | 一方的认知框架被摧毁 |
| 快感依赖 | ✗ | ✗ | △ | 剥夺快感来源或终结 |
| 身份本体 | ✗ | ✗ | ✓ | 暴力终结（身份不需要投降） |
| 意义否定 | ✗ | ✗ | ✗ | 选择而非论证（信仰一跃） |
| 结构性 | ✗ | ✗ | ✗ | 系统内部的自我瓦解 |

**Agent 使用方式**：

1. **设计反派时**：确定反派与主角的冲突属于哪种结构类型
2. **写对抗场景时**：确保冲突的"不可调和性"通过行为展示（不是通过旁白告知）
3. **审阅时**：检查反派是否在不该妥协的地方妥协了——如果是价值观存在论冲突，不能通过一次谈话化解

**不可调和冲突设计模板**：

```
反派 vs 主角冲突设计：
- 冲突类型：（从六种中选择 1-2 种主导）
- 不可调和的核心点：（一句话——为什么"理解"无法解决？）
- 反派需要从主角那里获得什么？（快感/身份确认/资源/消除威胁/其他）
- 主角不能放弃什么？（底线——放弃即否定自我）
- 如果双方"和平共处"，谁失去了什么？（至少一方失去存在意义 → 冲突不可调和）
- 可能的解决路径：（暴力终结/一方认知崩溃/系统瓦解/信仰一跃/其他）
```

## 五、Agent 心理分析工作流

### 5.1 「五更」：启发式交谈中的心理维度

在启发式交谈中新增以下可选询问维度：

- **角色心理状态**：本章各角色的心理状态是什么？三重人格中哪层在主导？处于面具稳固期/裂隙期/崩塌期？
- **情感弧线定位**：本章在整卷的情感弧线中处于什么位置？（上升/高潮/低谷/过渡）
- **心流参数**：本章冲突强度建议值？能力匹配度？
- **防御机制触发**：本章是否有角色会触发其防御机制？如何表现？
- **反派类型定位**（新增）：本章出场的反派属于哪种心理模板（§1.6）？A-F 哪一类？
- **恶意纯度**（新增）：反派的"不可理解"程度是多少？1-5 评分——1=完全可理解，5=完全不可调和
- **寒意级别**（新增）：本章反派场景的目标寒意级别（L1-L5）？是否需要启用寒意模式？
- **多样性检验**（新增）：引入新角色时，是否通过了七维多样性检验（§1.7）？

### 5.2 「琉璃」：心理一致性评审

在设定一致性评审中新增以下检查项：

| 心理检查项 | 检查内容 | 严重度 |
|-----------|----------|--------|
| **心理结构一致性** | 角色的行为是否与其已确立的三重人格结构一致？ | 🔴 |
| **防御机制一致性** | 角色面对压力时的反应是否匹配其主导防御机制？ | 🟡 |
| **原型行为一致性** | 角色的重大选择是否符合其核心原型的叙事逻辑？ | 🔴 |
| **情感递变合理性** | 角色在本章中的情感变化是否有足够铺垫？是否跳跃？ | 🟡 |
| **阴影揭示合理性** | 如果有阴影内容泄露，是否与角色的面具状态匹配？ | 🟡 |
| **成长弧线一致性** | 角色的心理变化是否符合已规划的成长弧线阶段？ | 🔴 |
| **反派心理一致性**（新增） | 反派行为是否在其犯罪心理模板（§1.6）的合理范围内？ | 🔴 |
| **恶意纯度一致性**（新增） | 已确定为"纯粹恶意"的反派是否出现了不符合该设定的"人性化"表达？ | 🔴 |
| **角色多样性审计**（新增） | 最近 5 章内出场的新角色是否覆盖了足够的多样性维度（§1.7）？ | 🟡 |
| **不可调和矛盾维持**（新增） | 已确立为不可调和的冲突（§4.4），是否出现了意外的调和/软化？ | 🔴 |
| **救赎弧线合理性**（新增） | 可救赎反派（B 类）的转变是否符合其心理模板的转变条件？代价是否足够？ | 🟡 |
| **寒意级别交付**（新增） | 反派场景实际交付的寒意是否达到目标级别（L1-L5）？ | 🟡 |

### 5.3 「墨」：心理描写生成指南

写角色内心戏时：

1. **不用情感形容词，用感官细节**（见第三章感官锚定系统）
2. **不用心理分析，用行为折射**（"他推了推眼镜，什么都没说" > "他内心翻涌着复杂的情感"）
3. **防御机制外化**：不写"他在压抑自己的愤怒"，写"他把桌上的文件一张张整理好，对齐边角，然后从头又整理了一遍"
4. **阴影泄露**：在角色独处/疲惫/醉酒/愤怒时，让阴影从面具缝隙中漏出——不是大段的黑暗面陈述，而是一句话、一个动作、一个不该有的笑

### 5.4 「墨」：反派场景生成指南（新增）

写反派场景时，启用以下行为准则：

**写作前**：
1. 确定反派所属模板（§1.6），读取对应模板的认知/情感/动机三层
2. 确定目标寒意级别（L1-L5），从五大寒意技法（§3.4）中选择 1-2 种
3. 如果是纯粹恶意反派（A/C/F 类）：禁止在反派内心独白中写"自我辩解"
4. 如果是可救赎反派（B 类）：确认当前处于转变弧线的哪个阶段

**写作中**：
1. **不用形容词告知反派的恐怖**——用行为展示。❌"他是个冷酷无情的人" ✅"他看着屏幕上的数字跳动。第17个。他没有任何表情变化。"
2. **反派的对话检查**：ta在"辩解"还是在"陈述"？纯恶意反派不辩解——ta的逻辑自洽，不需要你的认可
3. **移情屏障**：至少保留一处读者"试图共情但碰到透明墙"的细节
4. **反高潮的平静**：暴力/恐怖场景后，插入一个极日常的细节（不是对比——是并置）

**写作后**：
1. 寒意自检：场景是否达到了目标寒意级别？
2. 恶意纯度自检：纯粹恶意反派有没有不该出现的"人性温暖"？
3. 多样性自检：这个反派的作恶方式和之前出场的反派是否有足够差异？

## 七、常见心理叙事模式

### 7.1 "面具崩塌"场景公式

```
铺垫（面具在，但已有裂隙）→ 触发事件（恰好击中阴影）→ 
崩塌瞬间（不是大哭大闹——是一个不该有的沉默/笑/动作）→ 
后果（周围人的反应 + 角色自己的羞耻/释然）
```

### 7.2 "阴影整合"场景公式

```
对抗（主角 vs 阴影实体/内心黑暗面）→ 
承认（"是，这部分也是我"）→ 
转化（阴影不再是敌人，而是力量来源的一部分）
```

### 7.3 "防御机制失效"场景公式

```
压力升级（超出防御机制能应对的范围）→ 
旧机制产生反效果（让情况更糟）→ 
角色被迫面对自己绕不过去的东西 → 
新的应对方式诞生（或崩溃）
```

## 八、认知重构阶段模型

> 参考：梁元·月食 情感状态六阶段。此模型描述角色面临信念冲击时的渐进转变过程。用于写作前角色状态标注和评审时行为一致性检查。

### 8.1 六阶段定义

| 阶段 | 名称 | 特征 | 叙述约束 |
|------|------|------|---------|
| ① | 无自觉默认状态 | 按既有信念运作，未感知到冲击 | 正常行为，无特殊标记 |
| ② | 怀疑与否认 | 直觉上不信任新信息，觉得"有假" | 写角色找理由 dismiss，但不要说"他不相信" |
| ③ | 矛盾与撕扯 | 理智上未接受，但旧信念出现裂缝 | 写行为上的犹豫/反复，不要说"他纠结" |
| ④ | 顿悟与重构 | 某个具体时刻，新认知形成 | 写触发事件和身体反应，不要说"他终于明白" |
| ⑤ | 试错与检验 | 用新认知指导行动，测试有效性 | 写具体的试探行为，可能出现倒退 |
| ⑥ | 无自觉新状态 | 新信念内化，不再需要刻意维护 | 行为自然改变，不再提及旧信念 |

### 8.2 阶段转变规则

- 转变必须有**标志性事件**——读者能指出的具体时刻（某个动作/某句话/某个瞬间）
- 阶段不可跳跃——从②直接跳到④跳过③是不合法的（中间必须有撕扯）
- 不同角色的同一阶段表现不同——取决于其防御机制和人格结构
- **叙述中禁止直接说出阶段名称**——不写"他进入了矛盾与撕扯阶段"

### 8.3 阶段转变的标志类型

| 标志类型 | 示例 |
|----------|------|
| **行动标志** | 角色做了之前绝不会做的事 |
| **语言标志** | 角色说出了之前说不出口的话 |
| **身体标志** | 某种身体反应在特定情境下不再出现/第一次出现 |
| **关系标志** | 对他人的态度发生了可观察的改变 |
| **选择标志** | 在类似情境下做出了与之前不同的选择 |

### 8.4 Agent 使用方式

**角色创建/丰富时**：确定角色当前处于哪个阶段（针对哪个信念）。一个角色对不同信念可能处于不同阶段。
**写作时（mo-writer 简报 §2）**：标注角色的当前阶段 + 本章的预期阶段变化（维持/转变/倒退）
**评审时（ping-critic Step 3b）**：检查：
- 角色行为是否符合当前阶段的行为边界？
- 如果发生了阶段转变，是否有标志性事件？
- 转变是否跳过了必要的中间阶段？
**角色状态追踪（settings-manager record-character-state）**：在 `_character-state.md` 中记录每个角色的当前阶段 + 针对的信念。阶段转变时更新。

## 九、社会表演与面具

> 参考：梁元·月食 矜持化系统，基于 Goffman 拟剧论（The Presentation of Self in Everyday Life, 1956）。此框架描述角色在不同社会空间中的行为切换规律。

### 9.1 舞台区域

| 区域 | 定义 | 典型行为 |
|------|------|---------|
| **前台** (Front Stage) | 角色感知到"观众"在场的空间 | 注意形象、行为得体、维持社会身份、台词经过筛选 |
| **后台** (Back Stage) | 角色确信无观众的空间 | 放松、邋遢、放下防备、说真心话、做自己 |

**判断标准**：不是物理空间决定，而是角色**主观感知**的"是否有人在看"决定。

同一物理空间可以因人进入/离开而在前后台之间切换。角色的私人房间是后台，但有人推门进入的瞬间就变成了前台。

### 9.2 面具状态

- **Secure（面具在位）**：完美表演，无可见破绽。最多显得有点"排练过"，绝不慌乱。无论内心多汹涌，外部行为在角色控制之下。
- **Absent（面具卸下）**：无表演。开放、不设防、不在意给人什么印象。行为不受社会期望约束。

**关键规则**：面具不允许出现"裂缝"——切换是瞬时的，不存在半戴半卸的中间状态。角色不能"面具正在碎裂"——它要么在，要么不在。

### 9.3 切换触发条件

面具切换（Secure ↔ Absent）必须由以下三种触发之一引起：

1. **观众变化**：有人进入/离开当前空间（最常见）
2. **前后台转换**：角色从公共空间进入私密空间（或反过来）
3. **镇定阈值突破**：情绪冲击过大，维持面具的代价超过了维持它的能力

触发后切换是**瞬时的**——从一个状态直接跳到另一个，不允许渐进过渡。

### 9.4 Agent 使用方式

**写作前（mo-writer 简报 §1 场景设计）**：为每个场景标注：
- 舞台区域（前台/后台）
- 每个在场角色的面具状态（Secure/Absent）
- 场景中的面具切换点（谁在什么时候因什么触发切换）

**评审时（ping-critic Step 3b）**：检查：
- 角色的行为是否与其舞台区域一致？（后台中的角色不应表现得像在接受审查）
- 面具切换是否由合法触发条件引起？
- 切换是否瞬时完成？（"慢慢放下伪装"是错的——要么在伪装，要么没有）
- 切换后角色的行为是否与新的面具状态一致？

**交谈时（qing-novelist 场景设计维度）**：引导作者思考"这个场景谁是观众？角色在谁面前在表演？在谁面前可以做自己？"

## 十、心理动力模型统一框架

> **定位**：本文件包含多个心理学模型（弗洛伊德/荣格/Goffman/认知重构/防御机制/VAD）。此章定义它们之间的**主次关系和使用边界**——避免模型重叠导致的作者困惑和 Agent 判定不一致。

### 10.1 问题：模型重叠与冗余

当前角色模板的"心理动力结构"节要求作者填写四套独立框架：

```
三重人格（弗洛伊德）：本我 / 自我 / 超我
荣格原型：          核心原型 / 人格面具 / 阴影 / 面具崩塌
防御机制：          主导防御 / 触发场景 / 外在表现
情感基线：          VAD 数值
```

加上新增的认知重构六阶段（§8）和社会表演（§9），共计六套理论。它们之间存在大量概念重叠：

```
荣格 "人格面具"    ≈  Goffman "面具 Secure"
荣格 "阴影"        ≈  弗洛伊德 "本我"
荣格 "面具崩塌"    ≈  Goffman "镇定阈值突破"
弗洛伊德 "防御机制" ≈  Goffman "面具维持策略"
弗洛伊德 "超我"    ≈  Goffman "前台行为规则"
```

**实际使用数据**（Agent Skill 引用统计）：弗洛伊德被五更 29 次/琉璃 8 次/墨 4 次频繁引用；荣格被 0 个 Skill 引用——作者填写后无人读取。模型多不等于价值多——未被 Skill 使用的模型是装饰性字段。

### 10.2 统一原则：场景可操作性优先

这是一个**写作工程**，不是心理学工程。模型的唯一价值是"能否帮作者写出更好的场景"。选择标准：

1. **能否直接回答"角色在这一场景中应该如何表现"？** → 主线模型
2. **能否追踪角色的长期变化？** → 追踪模型
3. **能否提供角色创建的起点模板？** → 设计模型
4. **能否提供特定情境下的应对策略？** → 辅助工具

### 10.3 四层统一体系

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
第一层：社会表演框架（Goffman）—— 每次写场景的操作模型
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  回答："角色此刻在谁面前？在演还是在做自己？"
  
  前台/后台 → 决定整个场景的角色行为基调
  面具 Secure/Absent → 决定角色对外部世界的展示状态
  切换触发 → 决定场景中角色何时、为何突然改变行为模式

  为什么是主线：因为每一场戏角色都在某个空间里、面对某个人。
  空间+观众直接决定了角色能做什么、不能做什么。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  回答："在面具之下，角色对某个信念的接受到了哪一步？"
  
  六个阶段追踪从"旧信念稳固"到"新信念内化"的全过程。
  仅在角色面临信念冲击时激活——90% 的场景不需要此层。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
第三层：荣格原型 —— 仅用于角色创建时的起点模板
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  回答："这个角色最接近哪种经典形象？"
  
  选择一个原型作为"起点记忆锚"，填完即退出日常使用。
  不是每个场景都要回来查原型——原型是角色DNA，不是操作手册。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
辅助工具层
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  防御机制 → 面具受到威胁时的具体应对策略目录（"ta怎么保护面具？"）
  VAD      → 情感测量工具（"温度计"，不是"诊断书"——量化情绪状态）
  弗洛伊德 → 降级为 Skill 内部推理透镜（理解角色内部冲突时用，
             不在模板中要求作者逐层填写）
```

### 10.4 旧模型到新框架的映射

| 旧字段（分散在多处） | 新归属 | 说明 |
|---------------------|--------|------|
| 三重人格·本我 | → 社会表演·后台真实 | "ta独处时最原始的欲望"比"本我"更直觉 |
| 三重人格·超我 | → 社会表演·前台规则 | "ta在别人面前遵循什么规矩"比"超我"更直觉 |
| 三重人格·自我 | → 面具维持策略 | 自我=前台与后台之间的调解者 |
| 荣格·人格面具 | → 社会表演·前台形象 | 同一概念，Goffman 术语更易操作化 |
| 荣格·阴影 | → 社会表演·后台真实中被压抑的部分 | 阴影不是全部后台——是ta自己都不愿承认的那块 |
| 荣格·面具崩塌 | → 社会表演·切换触发(镇定阈值突破) | 同一现象，Goffman 明确了触发条件 |
| 防御机制 | → 面具维持策略 | 防御机制=面具受到威胁时的自动化应对 |
| 荣格·核心原型 | → 第三层：原型锚点 | 仅创建时使用，降为可选字段 |
| 情感基线 VAD | → 辅助：情感测量 | 保持独立——它是量化工具，不参与人格模型 |
| 认知重构六阶段 | → 第二层：信念转变 | 保持独立——它是过程追踪，不参与日常行为 |

### 10.5 角色模板的对应调整

统一后，角色模板的"心理动力结构"节从四套独立框架合并为一个以社会表演为主线的结构：

```
心理动力结构（以社会表演为主线）
  ├── 前台形象：        ta在公众/不熟的人面前是什么样？（← 原 荣格·人格面具 + 弗洛伊德·超我）
  ├── 后台真实：        ta独处/在最信任的人面前是什么样？（← 原 弗洛伊德·本我 + 荣格·阴影）
  ├── 面具切换触发：    什么情境ta会从前台跌入后台？（← 原 荣格·面具崩塌）
  ├── 面具维持策略：    ta用什么方式维持前台形象？（← 原 防御机制）
  ├── 当前认知阶段：    对关键信念各处于六阶段中的哪个？（← §8）
  └── 原型锚点：        创建时的荣格原型——仅作标签，不逐章参考（← 原 荣格·核心原型）
```

**弗洛伊德的退出**：id/ego/superego 不再作为作者填写字段。琉璃评审时如发现角色行为冲突，可以用"后台欲望 vs 前台规则"的框架来理解——这是 Goffman 的语言，不需要弗洛伊德的术语。

### 10.6 各 Skill 使用指南

| Skill | 何时用哪个模型 | 具体行为 |
|-------|--------------|---------|
| **mo-writer** (简报生成) | **主线：社会表演** | 为每个场景标注舞台区域 + 每个角色的面具状态 + 切换点。读认知阶段做角色弧标注。原型仅在新建角色时参考 |
| **ping-critic** (综合评审) | **主线：社会表演** | Step 3b 检查行为与舞台区域一致性、面具切换合法性。Step 3b 读认知阶段检查行为边界。防御机制仅在角色面临压力时检查 |
| **qing-novelist** (交谈) | **主线：社会表演** | 场景设计维度引导作者思考"谁是观众"。角色维度引导思考"ta在谁面前最放松"。认知阶段用于讨论角色弧 |
| **settings-manager** (状态追踪) | **主线：认知阶段** | `_character-state.md` 追踪每个角色的当前认知阶段 + 面具切换历史。VAD 用于情感峰值记录 |

### 10.7 荣格原型——降级但保留

荣格原型（英雄/阴影/阿尼玛/大母神/智慧老人/捣蛋鬼/自性）不参与日常写作和评审。但保留其在角色创建时的价值：

- **新建角色时**：从荣格原型中选一个作为"性格DNA的起点标签"。这是启发工具，不是分析工具。
- **角色出现行为不一致争议时**：琉璃可以回溯到原型锚点——"ta的原型是捣蛋鬼，捣蛋鬼会这样做吗？"——但这不是日常检查项。仅在有争议时回溯。
- **角色模板中简化**："核心原型"字段从必填降为选填，移到"原型锚点"下，一行即可。

### 10.8 降级规则

| 场景 | 行为 |
|------|------|
| 角色模板未填写"前台形象/后台真实" | 🚫 硬阻断——社会表演框架缺失则无法判断场景行为一致性 |
| 角色模板未填写"原型锚点" | ⚠️ 不影响功能——原型是创建启发，不是操作依赖 |
| 角色模板未填写"当前认知阶段" | ⚠️ 仅在涉及信念转变的场景中降级——日常场景不受影响 |
| 角色模板保留了旧字段（本我/自我/超我） | ℹ️ 兼容读取——琉璃可以将其映射到新框架（本我→后台真实，超我→前台规则）但不强制更新 |
