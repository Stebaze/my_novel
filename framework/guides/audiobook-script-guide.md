# AVG 剧本写作指南

> **定位**：当 `project-config.md` 中 `output_format=script` 时，指导如何在现有 novel 写作工作流下产出 AVG 剧本。
> 
> **格式规范**：完整的块类型语法定义见 `framework/guides/avg-script-format.md`。
> **立绘规格**：角色视觉呈现规范见 `framework/templates/characters/_sprite-spec.md`。

---

## AVG 剧本写作流程

`output_format=script` 时，写作工作流保持不变（plan-chapter → 五更交谈 → 墨简报 → 作者执笔 → chapter-review → publish-chapter），但产出物格式从叙事散文变为 AVG 剧本：

```
plan-chapter（规划 + 简报）
  └── mo-writer 阶段 6：从 _script-chapter-template.md 创建脚本章节文件
       ├── 预填：地点与氛围设计 / 出场角色与立绘差分 / 剧本正文脚手架
       └── 参考示例：sensory-writer 方法三生成 AVG 格式全章脚本示例
           │
           ▼
  【作者根据简报 + 脚本模板自己写 AVG 剧本】
           │
           ▼
chapter-review（评审）
  └── ping-critic 3c2：AVG 专项检查（立绘完整性/表情标注/演出注质量/bg prompt/CG prompt）
           │
           ▼
publish-chapter（发布）
  └── 从「剧本正文」提取 → 剥离脚手架 → 正式稿
```

## 格式基准

AVG 剧本使用 `framework/guides/avg-script-format.md` 定义的 11 种块类型：

| 类型 | 用途 |
|------|------|
| 场景头 | `【场景：地点·时间】` |
| 环境指令 | `【BGM：…】` / `【bg：背景ID】` |
| 立绘指令 | `【立绘：角色 表情 位置 [入场/退场]】` |
| 效果指令 | `【bgm 静默 N秒】` / `【停顿 N秒】` / `【特效：…】` |
| 对话行 | `角色名 [表情]：「台词」 ⏱ N.Ns` |
| 演出注 | 对话下方缩进，括号内——动作/语气/微表情 |
| 旁白块 | `【旁白】…【/旁白】` |
| 检定块 | `【检定：技能 \| 难度：NN \| 结果】`（仅跑团Replay VN） |
| CG块 | `【CG：名称 \| 触发时机】` + 生图提示词 |
| 选择块 | `【选择】` + 选项列表（仅标准AVG分支型） |
| 跳转指令 | `→ 场景_ID`（仅标准AVG分支型） |

**语法速查**见 `framework/templates/_script-format-spec.md`。

## 立绘管理

每个出场角色需要一份立绘规格卡（`novel/characters/{角色名}_sprite.md`），从 `framework/templates/characters/_sprite-spec.md` 模板创建。

**立绘规格卡包含**：
- **基础外观**：身高/发型发色/眼型瞳色/体型/标志性特征/整体氛围
- **表情清单**：从通用表情集中选择适用表情 + 角色专属自定义表情。每表情含视觉描述/典型触发场景/典型对白示例
- **服装变体**：制服/便服/礼装/战斗装束等
- **姿势变体**（可选）：标准站立/抱臂/靠墙/背对等
- **入场/退场偏好**：默认入场方式（滑入/淡入）

**在剧本中使用立绘**：
1. 角色首次出场 → `【立绘：角色 表情 位置 滑入】`
2. 角色情绪变化 → `【立绘：角色 新表情】` + 下句对白标注 `[新表情]`
3. 角色离开场景 → `【立绘：角色 淡出】`

**通用表情集**（13 种）：默认 / 微笑 / 苦笑 / 开心 / 生气 / 害羞 / 悲伤 / 眼眶泛红 / 惊讶 / 无语 / 坏笑 / 认真 / 逞强

## 散文→AVG剧本转换对照

### 散文（prose 模式）

> 圣光帷幕在许心照身前展开的那一瞬间，林衣的刀已经到了。银白色的刀光如同一道月华凝成的匹练，狠狠地斩在了那纯白无暇的屏障之上。太阴之力与圣堂的庇佑之光在半空中激烈碰撞，发出刺耳的尖鸣——那是两种截然不同的力量体系在互相侵蚀。
> 
> 林衣眉头微挑。这一刀没有斩破。
> 
> "这屏障有意思，不是纯粹的灵力护盾，里面混了别的东西。"
> 
> "还没完呢。"许心照的声音从帷幕后方传来，语气依旧是那股元气满满的调子，但底下压着认真。

### AVG 剧本（script 模式）

```
【场景：模拟秘境·演武场】
> 场景总时长：约 2分30秒（15句对白 + 3处停顿）
【BGM：紧张弦乐——低音部持续，偶有打击乐点】
【bg：秘境_演武场】

【旁白】
圣光帷幕在许心照身前展开的瞬间，林衣的刀已经到了。银白色刀光如月华凝成的匹练，斩在纯白屏障之上——太阴之力与圣堂庇佑之光激烈碰撞。
【/旁白】

【音效：力量对撞——刺耳尖鸣，两种能量互相侵蚀】
【立绘：林衣 默认 左侧】
【立绘：许心照 默认 右侧】

林衣 [默认]：「这屏障有意思。不是纯粹的灵力护盾，里面混了别的东西。」 ⏱ 3.5s
  （眉头微挑，目光在屏障表面扫过——不是在赞叹，是在分析）

【立绘：许心照 认真】

许心照 [认真]：「还没完呢。」 ⏱ 1.2s
  （帷幕后方传来。语调依旧是那股元气满满的调子，但底下压着认真）
```

**核心差异**：
1. 散文的叙述 → 拆成旁白块 + 音效 cue + 演出注
2. 散文的对话标签（"许心照的声音从帷幕后方传来"）→ 演出注
3. 散文的心理活动（"显然意识到了什么"）→ 删除或移入演出注（"不是在赞叹，是在分析"）
4. 散文的连续段落 → 结构化的块序列（场景头→BGM→bg→旁白→立绘→对话）
5. 每个情绪变化 → 立绘指令更新
6. 每句台词 → `[表情]` + `⏱ N.Ns`

## 项目类型与块类型选用

`project-config.md` 需额外配置项目类型，决定哪些块类型可用：

### 标准 AVG（分支型）

| ✅ 使用 | ❌ 不使用 |
|---------|----------|
| 场景头 / BGM / bg / 立绘 / 对话 / 演出注 / 旁白 / CG / 静默停顿 / **选择块** / **条件块** / **跳转指令** / **Flag标记** | 检定块 |

### 跑团 Replay VN（线性型）

| ✅ 使用 | ❌ 不使用 |
|---------|----------|
| 场景头 / BGM / bg / 立绘 / 对话 / 演出注 / 旁白 / CG / 静默停顿 / **检定块** | 选择块 / 条件块 / 跳转指令 / Flag标记 |

默认 `output_format=script` 时使用**跑团 Replay VN**（线性型）——与现有 novel 写作工作流的线性章节推进一致。
