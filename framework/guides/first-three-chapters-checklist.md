# 前三章验证清单

> **使用时机**：每章写完初稿后逐条打勾。任意一章超过 3 个 ❌ 就需要重写。
> **核心理念**：前三章不是"写得怎么样"的问题，而是"读者留不留"的问题。

---

## 第一章验证

### 开幕（前 100 字）

- [ ] 读者读完 100 字后，知道主角叫什么吗？
- [ ] 读者读完 100 字后，知道主角在做什么吗？
- [ ] 前 100 字里有什么让人想继续读的东西吗？（一句有趣的话/一个反常的画面/一个疑问）

### 主角建立

- [ ] 读完第一章后，能用一句话描述主角是什么人吗？（不是"他是XX身份"，而是"他是一个XX的人"）
- [ ] 主角在这一章里有至少一次主动行为吗？（不是"事情发生在主角身上"，而是"主角做了某件事"）
- [ ] 主角有什么让人记住的特征？（一个动作习惯/一种说话方式/一个执念）

### 世界建立

- [ ] 读者知道这个故事发生在什么样的世界里吗？（现实日常？有系统的现实？架空？）
- [ ] 世界设定是通过场景/对话自然带出的，还是通过大段旁白介绍的？
- [ ] 如果作品有系统/金手指，读者知道它的基本运作方式吗？（不需要全懂，但要知道"有个东西在运作"）

### 钩子

- [ ] 第一章结尾时，有一个明确的问题/悬念让读者想点下一章吗？
- [ ] 这个问题是关于"人"的（角色的处境/关系/秘密），而不是关于"设定"的（"这个世界是怎么回事"）？

**第一章总结**：❌ 数量 = ___ / 10（≥3 需要重写）

---

## 第二章验证

### 能力/系统展开

- [ ] 读者更了解主角有什么独特之处了吗？（比第一章更具体）
- [ ] 主角使用了他的能力/系统吗？（不能只是"介绍"，要"展示"）
- [ ] 如果有代价/副作用，读者看到代价的苗头了吗？

### 第一个冲突

- [ ] 这一章有一个让主角感到压力的冲突/问题吗？
- [ ] 这个冲突是主角自己选择面对的，还是被动卷入的？
- [ ] 冲突结束时，局势比开头更紧张了吗？

### 节奏

- [ ] 这一章至少有一次"爽点"或"甜点"吗？
- [ ] 如果第二章还没引入主要女性角色，那有什么替代的情感锚点？

**第二章总结**：❌ 数量 = ___ / 6（≥2 需要重写）

---

## 第三章验证

### 女性角色引入

- [ ] 这一章里有一个女性角色让人记住她吗？（记住 = 一个特征 + 一个台词 + 一个动作）
- [ ] 她和男主的第一次互动有什么"特殊之处"？（不是普通的打招呼/介绍——有没有尴尬/误会/反差/名场面？）
- [ ] 她的说话方式和其他角色区分得开吗？（遮住名字能认出来吗？）

### 基调确立

- [ ] 读者知道这本书的"味道"是什么了吗？（读完后是什么感觉——搞笑/甜/燃/虐/治愈？）
- [ ] 前三章的整体情绪走向是上升趋势吗？（越来越想读，不是越来越平淡）

### 追读意愿测试

- [ ] **你找一个不认识你的人读前三章，第三章结束时 ta 会问"然后呢？"吗？**
- [ ] 前三章中出现过至少一次让人想"截图分享"的台词或画面吗？

**第三章总结**：❌ 数量 = ___ / 6（≥2 需要重写）

---

## 前三章整体检查

- [ ] 三章的时间跨度是否合理？（不能三章都在同一天同一场景）
- [ ] 每章的结尾是否都是有效的断章？（卡高潮/悬念/情感揭示——不是自然结束）
- [ ] 三章中是否出现过至少一次"日常"场景让角色放松也让读者喘息？
- [ ] 如果这是读者看的唯一三章——他会觉得"这本书在讲一个什么样的故事"？这个印象对吗？

**整体 ❌ 数量 = ___ / 4**

---

## 结论

- 各章单独 ❌ 均未超标，且整体 ❌ ≤ 1 → ✅ 可以继续写第四章
- 某章 ❌ 超标 → 🔴 该章需要重写
- 整体 ❌ 超标但各章 OK → 🟡 问题可能在章节之间的衔接，检视过渡
