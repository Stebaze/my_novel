# 荣格分析心理学 —— 角色心理动力操作框架

> **定位**：本文档是角色心理动力的统一操作框架。以荣格分析心理学为主线，将其他心理学模型（Goffman 社会表演、弗洛伊德防御机制、认知重构阶段、VAD 情感模型）映射进荣格的概念体系。
>
> **核心立场**：这是一个**写作工程**文档，不是心理学教材。每个概念必须翻译为场景中可观察的角色行为。理论只在能帮作者写出更好的场景时才有价值。
>
> **与 psychology-guide.md 的关系**：本文档是**操作手册**——告诉 Skill "做什么"；`psychology-guide.md` 是**参考百科**——提供理论背景"为什么"。操作手册独立可用，不需要查阅百科。

---

## 一、核心原则

### 1.1 荣格的核心洞见

人有一个"成为自己"的天然方向。荣格称之为**自性化**——将人格面具（你在他人面前扮演的角色）与阴影（你隐藏、否认或尚未意识到的部分）整合为完整自我的过程。

对角色写作而言：**角色的成长弧线就是自性化的过程。** 每一场戏都在问：这个角色此刻离"完整的自己"更近还是更远？

### 1.2 场景可操作性优先

心理学概念必须回答"作者在这一场景中怎么写这个角色"。不能回答这个问题的概念不进入操作框架。

### 1.3 框架结构

```
人格面具 ←──张力──→ 阴影
    │                  │
    └──── 自我调解 ────┘
              │
              ▼
         自性化（六阶段成长弧线）
              │
              ▼
         内在驱动（每场戏的行为引擎）
              │
              ▼
         情感基调（VAD 量化）
```

---

## 二、人格面具

> **一句话定义**：人格面具是角色在他人面前展示的社会身份——不是"真实的自己"，而是"角色希望别人认为的自己"。

### 2.1 面具的定义与功能

人格面具不是伪装——它是角色在社会中生存的必要工具。每个人都有面具，没有面具的人无法在社会中行动。面具的叙事功能：

- **角色辨识度**：读者通过面具快速认识角色（"冷傲剑客""话痨死党""温柔师姐"）
- **行为预期**：面具提供角色在社交场景中的默认行为模式
- **冲突源**：当面具与内在真实需求冲突时，产生戏剧张力

### 2.2 面具的激活与放松

| 状态 | 对应概念 | 定义 | 典型行为 |
|------|---------|------|---------|
| **面具激活** | 前台 | 角色感知到"观众"在场 | 注意形象、行为得体、台词经过筛选 |
| **面具放松** | 后台 | 角色确信无观众 | 不再筛选行为、说真心话、放下社交姿态 |

判断标准：不是物理空间决定，而是角色**主观感知**的"是否有人在看"决定。同一个房间可以因人进入而从后台变为前台。

### 2.3 面具的强度

面具不是二元开关——它有不同的强度级别：

| 强度 | 定义 | 何时出现 |
|------|------|---------|
| **稳固** | 面具完全在位，表演无懈可击 | 公开场合、陌生人面前、角色有意识维持形象时 |
| **松动** | 面具仍在，但出现细小裂缝——一句话没收住、一个表情没控制好 | 疲劳时、情绪波动时、面对特定人物时 |
| **卸下** | 面具完全放下 | 独处时、在最信任的人面前、情绪彻底崩溃时 |

**关键规则**：面具从稳固到卸下的转变是**瞬时的**——不存在"半戴半卸"的中间状态。"面具正在碎裂"是错的——它要么还在，要么已经不在了。松动是裂缝，不是半卸。

### 2.4 面具切换触发条件

面具强度变化必须由以下触发引起：

1. **观众变化**：有人进入/离开当前空间（最常见）
2. **空间转换**：从公共空间进入私密空间（或反过来）
3. **情绪阈值突破**：情绪冲击过大，维持面具的代价超过了继续维持的能力

### 2.5 面具与场景设计

**写作前**：为每个场景确定——舞台区域（前台/后台）+ 每个在场角色的面具强度 + 场景中可能的面具切换点。

**写作中**：角色的台词、肢体语言、内心活动必须与其面具强度一致。面具激活时角色不会说出真心话——但可以通过肢体语言或内心独白泄露。

**评审时**：检查——角色的行为是否与其面具强度一致？面具切换是否由合法触发引起？切换是否瞬时完成？

## 三、阴影

> **一句话定义**：阴影是角色隐藏的、否认的、或尚未意识到的心理内容——被压抑的欲望、不敢承认的恐惧、与面具形象冲突的特质。

### 3.1 阴影的来源与内容

阴影包含：
- 角色**主动压抑**的内容（因为不符合面具形象或社会期望）
- 角色**尚未意识到**的内容（天生的盲区）
- 角色**否认**的内容（"我不是那种人"——但其实是）

阴影不是"邪恶"——它只是"被拒绝"。一个温柔角色可能压抑了愤怒；一个强大角色可能压抑了脆弱。

### 3.2 阴影的表达方式

阴影不能一直被压着。它通过以下方式表达——这些是作者写阴影的工具：

| 表达方式 | 定义 | 叙事使用 |
|----------|------|---------|
| **投射** | 角色在他人身上看到自己否认的特质 | "我不是怕——是你怕了" |
| **情绪失控** | 在极端压力下阴影冲破控制 | 突然暴怒/崩溃大哭/说出一直不敢说的话 |
| **独处泄露** | 独处时做出与面具形象不符的行为 | 冷酷角色独自时看着旧照片发呆 |
| **身体信号** | 阴影通过身体表达——角色意识不到但读者能看到 | 嘴上说没事，手在发抖 |
| **梦境/幻觉** | 阴影在意识薄弱的时刻浮现 | 梦中出现反复的意象、幻觉中看到自己 |

### 3.3 阴影与人格面具的张力——核心戏剧冲突

每个角色的核心戏剧冲突根植于**人格面具与阴影之间的张力**：

```
面具："我是一个冷静、理性、不会被打倒的人"
阴影："我需要被爱。我怕被抛弃。我小时候被伤害过。"

张力 = 角色在多大程度上能承认阴影的存在
```

这个张力是角色所有重大决策的真正来源。剧情不是"发生"在角色身上——是角色因为面具与阴影的冲突而主动做出的选择推动的。

### 3.4 阴影的叙事功能

- **制造矛盾**：阴影冲动与面具规则的冲突 → 角色做出意料之外的选择
- **解释盲区**：角色为什么在某个问题上反复犯错 → 阴影中有ta不愿面对的真相
- **驱动改变**：角色只有在面对阴影时才能成长 → 自性化的起点

### 3.5 阴影的极端形态——反派

> 反派不是"有阴影的角色"——反派是**阴影吞噬了人格面具**的角色。ta 不再"表演"社会接受的形象，阴影成为其行动的唯一驱动。

**反派与常规角色的阴影差异**：

| 维度 | 常规角色 | 反派 |
|------|---------|------|
| **阴影地位** | 被压抑、否认 | 被接纳、甚至被崇拜 |
| **人格面具** | 在位（有松动的可能） | 已崩溃或已与阴影同化 |
| **自性化方向** | 整合阴影 → 成为更完整的人 | 拒绝整合 → 阴影成为"我就是这样的人"的终极辩护 |
| **对读者的效果** | 共情（"我也怕这个"） | 寒意（"我可能也会变成这样"） |

**反派阴影的三种叙事形态**：

1. **阴影外化型**：反派的阴影与主角的阴影是同一个内容的不同表达。主角压抑的，反派活了出来。反派是主角"如果不回头会变成的样子"。
   - 写法：让反派在某场戏中说出主角内心不敢说的话，或做出主角暗自想过但不敢做的选择。
   
2. **阴影合理化型**：反派为自己的阴影构建了一套逻辑自洽的辩护体系。ta 不是不知道自己有问题——ta 真诚地认为"问题其实是世界的，不是我的"。
   - 写法：给反派一个"如果只看 ta 的视角，确实有道理"的信念内核。读者在某个瞬间会想"ta 说的也对"——这就是寒意。

3. **阴影空洞型**：反派的阴影内容本身就是"空"——没有可追溯的心理原因，没有可理解的动机。ta 的恐怖恰恰在于不可理解。
   - 写法：不给解释。其他角色（和读者）的恐惧来自"我理解不了 ta 为什么这样做"。用留白代替心理分析。

**使用**：创建反派时，先确定属于哪种阴影形态 → 对照 `framework/guides/villain-design-guide.md` §二 选择具体模板 → 填写角色档案的人格面具/阴影字段。审阅反派场景时，检查"ta 的阴影表达是否与其阴影形态一致"。

### 3.6 阴影驱动的场景与矛盾设计

> 阴影不只是角色属性——它是场景设计的引擎。每一个有张力的场景背后，都是一个角色的阴影在推。

**场景设计的阴影四问**（构思每场戏时使用）：

```
Q1: 这场戏中，谁的阴影最接近表面？
    → 找出"最不稳定"的角色——ta 的面具最松，阴影最可能溢出

Q2: 什么会触发这个阴影？
    → 一句台词？一个相似情境？一个特定人物的出现？一个物品？
    → 这个触发条件就是场景的"炸弹引信"

Q3: 阴影溢出后会以什么方式表达？
    → 投射（怪罪别人）？情绪失控？身体信号？独处后的泄露？
    → 选择表达方式 = 选择场景的情感质感和强度

Q4: 其他角色如何反应？
    → 谁的阴影被这个反应激活了？（连锁反应——一个人的阴影触发另一个人的）
    → 这制造了场景的"第二波张力"
```

**矛盾设计的阴影三模式**：

| 模式 | 结构 | 适用场景 |
|------|------|---------|
| **面具碰撞** | 角色A的面具行为规则 与 角色B的面具行为规则 互不相容 | 社交冲突、权力博弈、价值观对立 |
| **阴影互照** | 角色A在角色B身上看到了自己否认的阴影内容（投射）→ 被激怒/被吸引 | 角色关系的质变时刻、理解与误解 |
| **阴影链式反应** | 角色A阴影溢出 → 触发角色B的面具维持策略 → 角色B策略显得可疑 → 触发角色C的阴影 | 群戏中的多米诺效应、危机升级 |

**使用**：
- **写作前（mo-writer 简报 §1）**：为每个场景标注"阴影四问"的答案——哪个角色最不稳、触发条件是什么、预期表达方式、连锁反应
- **写作中（作者）**：场景的核心戏剧不是"发生了什么事"——是"谁的阴影在推"
- **评审时（ping-critic Step 3b）**：检查场景中阴影表达的一致性——角色的阴影溢出方式是否与其阴影内容匹配？触发条件是否合理？

## 四、自我与防御

> **一句话定义**：自我是面具与阴影之间的调解者——它决定什么可以表达、什么必须压抑、如何维持"我"的完整感。

### 4.1 自我的调解功能

自我承担三个任务：
1. **维持面具**：确保角色在社交场合不"失态"
2. **压抑阴影**：防止阴影内容冲破控制
3. **整合信号**：当面具与阴影的冲突变得不可持续时，发出"需要改变"的信号（焦虑、不适、梦）

### 4.2 面具维持策略

当面具受到威胁时，自我自动启动维持策略。以下策略目录供写作时参考：

| 策略 | 定义 | 外在表现（读者能看到/听到的） |
|------|------|------------------------------|
| **压抑** | 将威胁性内容推回无意识 | 突然沉默、转移话题、说"我不想想这个" |
| **合理化** | 用逻辑解释掉情感威胁 | "我不是怕——是理性评估后觉得不值得冒险" |
| **投射** | 把自己否认的特质归到他人身上 | "你才是在害怕的那个" |
| **反向形成** | 表现出与真实感受相反的行为 | 越喜欢越冷淡、越在意越装作无所谓 |
| **退行** | 回到更早的发展阶段的行为模式 | 成年角色突然像小孩一样发脾气 |
| **升华** | 将阴影冲动转化为社会认可的行为 | 攻击冲动 → 竞技体育 |

**选择原则**：每个角色有 1-2 种主导策略——这是ta的"默认武器"。不要给角色配满所有策略——那会让ta的心理防御看起来像瑞士军刀而不是性格特征。

### 4.3 策略失效的后果

当面具维持策略无法继续压住阴影：

1. **面具裂缝**（松动级）：一句没收住的真话、一个没控制好的表情
2. **阴影侵入**（卸下级）：阴影内容直接溢出——暴怒/崩溃/说出真相
3. **面具崩溃**：角色暂时失去了"表演"的能力——这是自性化的危机也是转机

## 五、原型

> **一句话定义**：原型是人类共有的心理模式。在写作中，它们提供角色设计的起点模板——填完即退出，不逐章参考。

### 5.1 常用原型速查

| 原型 | 核心命题 | 典型行为 | 常见角色类型 |
|------|---------|---------|------------|
| **英雄** | 证明自己的价值 | 主动出击、不服输、保护弱者 | 主角、冒险者 |
| **阴影** | 被压抑的黑暗面 | 诱惑、破坏、揭露虚伪 | 反派、反英雄 |
| **阿尼玛/阿尼姆斯** | 与异性的内在连接 | 情感敏感、直觉判断 | 知己、爱情对象 |
| **大母神** | 滋养与吞噬 | 照顾与保护、过度控制 | 导师、母亲角色 |
| **智慧老人** | 提供指引与真相 | 神秘、说谜语的话、关键时出现 | 导师、老前辈 |
| **捣蛋鬼** | 打破规则、制造混乱 | 不按常理出牌、反讽、恶作剧 | 喜剧角色、催化剂 |
| **自性** | 完整与整合 | 接纳自己的全部面向 | 角色成长终点 |

### 5.2 原型的使用规则

**创建时用一次**——选一个最接近的原型作为角色的"性格DNA起点标签"。填完「原型锚点」字段即退出，不进入日常写作和评审。

**出现争议时回溯**——当评审中角色行为出现一致性争议时，可以回溯到原型锚点："ta的原型是捣蛋鬼，捣蛋鬼在此情境下更可能打破规则还是遵守规则？"——但这不纳入日常检查项。

### 5.3 原型锚点的填写

一行即可：`原型锚点：英雄型——驱动来源：需要被认可`

不需要展开分析，不需要对照原型检查清单。这是记忆锚，不是操作手册。

## 六、自性化

> **一句话定义**：自性化是角色整合人格面具与阴影，成为"完整的自己"的过程——这是角色成长的底层引擎。

### 6.1 自性化的核心

自性化不是"变得更好"——是**变得更完整**。角色不需要消灭阴影——需要承认它、接纳它、让它的能量为意识所用。

对写作而言：角色的成长弧线 = 面对阴影 → 松动面具 → 整合 → 新平衡。

### 6.2 自性化的六阶段

> 此六阶段模型用于追踪角色对某个信念的转变过程。一个角色对不同信念可能处于不同阶段。

| 阶段 | 名称 | 角色行为特征 | 写作约束 |
|------|------|------------|---------|
| ① | **无自觉默认** | 按既有信念行动，未感知到问题 | 正常行为，无特殊标记 |
| ② | **怀疑否认** | 遇到与信念冲突的信息 → 直觉拒斥 → 找理由 dismiss | 写角色找借口的台词和行动，禁止写"ta不相信" |
| ③ | **矛盾撕扯** | 理智还撑着旧信念，但行为和身体已经动摇 | 写行为上的犹豫/反复，禁止写"ta纠结" |
| ④ | **顿悟重构** | 某个具体时刻，新认知形成 | 写触发事件+身体反应，禁止写"ta终于明白" |
| ⑤ | **试错检验** | 用新认知指导行动 → 测试其有效 → 可能有倒退 | 写具体试探行为，允许又缩回去 |
| ⑥ | **新平衡** | 新信念内化，不再需要刻意维护 | 行为自然改变，不再提及旧信念 |

### 6.3 阶段转变规则

- 转变必须有**标志性事件**——读者能指出的具体时刻（某句话、某个动作、某个瞬间）
- **不可跳阶段**——从②直接到④（跳过撕扯）是不合法的
- 不同角色的同一阶段**表现不同**——取决于其防御策略和人格结构
- **叙述中禁止直说阶段名称**——不写"他进入了矛盾与撕扯的阶段"

## 七、内在驱动

> **一句话定义**：荣格认为心灵是有目的性的——心理能量总是朝着某个目标流动。内在驱动八维将这个"目标方向"翻译为场景中可操作的角色行为参数。

### 7.1 心灵的目的性

角色不是"被动等待剧情发生"。在每一个场景开始时，ta 已经在做某件事、想要某个东西、朝着某个方向。内在驱动八维将这种"已经在流动的心理能量"捕获为具体参数。

### 7.2 八维驱动推导链

从远到近、从大到小地推导角色在此刻的行为驱动：

| 维度 | 问题 | 注意事项 |
|------|------|---------|
| **对他人认知** | 对其他角色的假设——身份/历史/气质/本质？可能正确或错误 | 决定了 ta 如何预判他人行为 |
| **人生终极目标** | 最想要什么？具体到能拍出一个画面 | 不是"变强"——是"成为能让师父说出'你出师了'的人" |
| **近期具体目标** | 数周/数月内可以"完成"或"失败"的事项 | 与人生目标有可追溯的路径 |
| **今日客观日程** | 今天本来要做什么——在剧情发生之前 | 锚定角色的日常基线 |
| **此刻正在做** | 场景开始前已在进行的动作 | 不能是"等待"——必须是一个具体动作 |
| **此刻的内在冲动** | 此刻最自私、主动的欲望——不是对剧情的反应 | 身体/情感/认知/社交四种驱动来源 |
| **冲动评估** | 与既有设定矛盾？优先级？自助 or 求助？ | 可选的——冲突时才需要 |
| **具体行动** | 为此冲动采取的外部可观察动作 | 读者必须能看到 |

### 7.3 驱动更新

每次场景结束后，角色的 At Hand / Urge 可能改变。自性化阶段可能在关键场景后发生转变。这些变更通过 `settings-manager` (record-character-state) 记录到 `_character-state.md`。

## 八、情感基调

> **一句话定义**：荣格用"情感基调"(feeling-tone)描述被情结围绕的心理内容携带的情绪色彩。VAD 三维模型是这个概念的量化工具。

### 8.1 情感基调与 VAD

| VAD 维度 | 含义 | 写作应用 |
|----------|------|---------|
| **Valence（愉悦度）** | 积极 ↔ 消极 | 角色对当前情境的情感态度 |
| **Arousal（唤醒度）** | 兴奋 ↔ 平静 | 角色的能量水平 |
| **Dominance（支配度）** | 掌控 ↔ 被掌控 | 角色的权力感 |

### 8.2 使用方式

**VAD 是测量工具，不是设计工具。** 用它来：
- 标记角色的情感起点和终点（本章的 VAD 变化）
- 检查情感递变是否合理（跳跃过大 = 缺少过渡）
- 追踪自性化过程中的情感趋势（如 Valence 逐渐从负到正）

不要用 VAD 来"设计"情感——用面具与阴影的张力来驱动情感，用 VAD 来量化验证。

## 九、角色模板字段映射

旧模板的"心理动力结构"节使用弗洛伊德+荣格混合术语。以下为旧字段→新字段的映射：

| 旧字段 | 新归属 | 说明 |
|--------|--------|------|
| 三重人格·本我 | → 阴影 | "ta 独处时最原始的冲动"属于阴影内容 |
| 三重人格·超我 | → 人格面具·行为规则 | "ta 在他人面前遵循什么规矩"是面具的一部分 |
| 三重人格·自我 | → 自我与防御 | 调解功能保留，归入面具维持策略 |
| 荣格·人格面具 | → 人格面具 | 名称不变，操作化增强 |
| 荣格·阴影 | → 阴影 | 名称不变，表达方式明确化 |
| 荣格·面具崩塌 | → 面具切换触发 | 统一为切换触发条件之一（情绪阈值突破） |
| 荣格·核心原型 | → 原型锚点 | 降为选填，仅创建时使用 |
| 防御机制 | → 面具维持策略 | 功能不变，表述从病理→功能 |
| 情感基线 VAD | → 情感基调 | 保持不变 |
| 内在驱动八维 | → 内在驱动 | 保持不变 |

**兼容读取规则**：如果角色档案仍有旧字段名（本我/自我/超我），Skill 可以兼容读取并映射到新框架（本我→阴影原始冲动、超我→面具内化规则）。不强制作者立即迁移已有角色档案。

## 十、各 Skill 使用指南

### 10.1 墨（mo-writer）—— 简报生成

**读取**：§2 人格面具（场景面具标注）、§3.5（反派阴影形态，涉及反派时）、§3.6（阴影四问——场景矛盾设计）、§6 自性化（当前阶段）、§7 内在驱动（角色入场状态）

**产出**：
- 简报 §1 场景设计：为每个场景标注舞台区域 + 各角色的面具强度 + 预期切换点 + 阴影四问答案（最不稳定的角色/触发条件/预期表达/连锁反应）
- 简报 §2 角色出场设计：标注每个角色的当前自性化阶段 + 内在驱动 + 本章阴影表达机会。反派角色标注其阴影形态（外化/合理化/空洞）+ selected 模板编号

**不读**：§5 原型（除非是新建角色）

### 10.2 琉璃（ping-critic）—— 综合评审

**读取**：§2 人格面具（Step 3b 一致性）、§3 阴影（Step 3b 阴影表达）、§3.5（反派阴影形态，涉及反派时）、§3.6（阴影四问——场景矛盾是否由阴影驱动）、§4 面具维持策略（Step 3b 防御行为）、§6 自性化（Step 3b 阶段一致性）

**检查项**（Step 3b 角色与设定层）：
- 面具一致性：角色行为是否与其面具强度一致？面具切换是否有合法触发？
- 阴影表达：角色的阴影是否以合理方式呈现？是否有未铺垫的阴影内容突然出现？反派的阴影形态是否一致（外化型反派不应突然空洞化）？
- 场景矛盾来源：场景的核心冲突是否可以追溯到某个角色的阴影驱动？还是仅为外部事件堆叠？
- 防御行为：压力场景中，角色是否使用了其主导面具维持策略？
- 自性化阶段：如发生阶段转变，是否有标志性事件？是否跳过阶段？

**不读**：§5 原型（除非角色行为出现根本一致性争议）

### 10.3 五更（qing-novelist）—— 启发式交谈

**读取**：§2 人格面具（维度 6 张力讨论）、§3 阴影（维度 6 张力方向）、§6 自性化（角色弧讨论）

**维度 6 交谈用语**：
- 旧："三重人格中哪层在主导？"
- 新："ta 的人格面具与阴影此刻在往哪个方向拉？ta 离面对自己的阴影更近还是更远了？"

**不读**：§4-5（防御策略和原型不进入交谈——交谈聚焦张力感知）

### 10.4 settings-manager —— 状态追踪

**读取**：§6 自性化（阶段追踪字段）、§7 内在驱动（驱动更新字段）、§8 VAD（情感峰值记录）

**record-character-state 追踪**：
- 自性化阶段：每个角色当前阶段 + 针对的信念
- 面具切换历史：本章是否发生面具切换？触发原因？
- 内在驱动更新：At Hand / Urge 的变化
- 情感峰值 VAD

### 10.5 pre-flight-check —— 前置检查

**C5b 角色人设完整度检查**：维度 4（心理动力结构）从检查"三重人格/荣格原型/防御机制"改为检查以下字段的存在性：
- 人格面具（面具形象 + 面具行为规则）
- 阴影（阴影内容 + 阴影侵入方式）
- 面具维持策略（主导策略）
- 自性化阶段（当前针对的信念 + 当前阶段）

**缺失处理**：人格面具或阴影任一缺失 → 🟡 角色单薄警告。面具维持策略或自性化阶段缺失 → ⚠️ 降级（不阻断，标注后继续）。
