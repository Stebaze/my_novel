# 参考素材库

> **定位**：本文档收录了从多部参考作品中系统提取的写作技法、大纲模式、角色原型、叙事技法、声音类型和词汇风格。原分散存放于 `framework/templates/` 各模板附录中，现集中于此作为独立的参考指南。
>
> **使用方式**：
> - 写作前：查阅相关章节获取技法灵感和结构参考
> - 启发式交谈中：根据对标作者快速查找其技法特征  
> - 同步机制：`workspace-sync` Skill 的 `sync-enrich` 操作会将新增的参考条目传导到 `novel/` 实例文件中
>
> **与模板的关系**：`framework/templates/` 中的模板文件仅保留 🏠 内置条目（来自方法论本身的通用技法）和空白占位符。本文档中的条目是在具体参考作品分析中提取的、标注了来源的参考技法。

## 一、大纲结构模式（6 种）

> 原存放于 `framework/templates/outline.md` 附录 A。以下 6 种结构模式提炼自多部作品，选择最接近你作品类型的 1-2 种作为结构参考。

### 模式 1：文明级渐进弧（远瞳型）
**来源**：《黎明之剑》《深海余烬》
**结构特征**：
- 每 8-15 章构成一个完整弧：发现异常 → 调查 → 发现更深层谜团 → 被迫选择立场 → 召集力量 → 决战 → 余韵（世界观揭示）
- 弧与弧之间是"嵌套"关系——不是"A 解决后开始 B"，而是"A 解决了一半，发现 B 才是真正的问题"
- 每弧结束时至少揭示一层"更大的世界观"
**L1 顶层设计**：终点是一个"文明级"的抉择或认知颠覆——不是"打败了谁"而是"理解了什么"
**适用**：奇幻史诗/文明建设/多势力博弈/渐进揭示型世界观

### 模式 2：目标驱动弧（言归正传型）
**来源**：《我师兄实在太稳健了》
**结构特征**：
- 以"目标"而非"章数"为单位——每个目标从发现到解决构成一弧，长度取决于目标复杂度
- 每弧的固定节奏：日常平静 → 发现异常细节（触发推演） → 制定多方案 → 意外打破计划 → 备用方案启动 → 备用方案也失效 → 临时推演+正面行动 → 战后复盘
- 弧之间有"风险层级递增"——每弧解决的威胁比上一弧更大
**L1 顶层设计**：终点是主角从"小心翼翼求生存"到"有能力保护想保护的人"的转变
**适用**：修仙/策略/多势力博弈/成长型主角

### 模式 3：体制生存弧（熊狼狗型）
**来源**：《没钱修什么仙？》
**结构特征**：
- 以"比赛/考试/晋升"为弧骨架：赛前准备 → 对手/环境展示 → 比赛中利用规则 → 意外出现 → 生存策略调整 → 结果（不一定是赢）
- 每弧揭示一层"更荒诞的规则"——适应一个小荒诞后遇到一个更大的
- 弧的爽点不是"打败所有人"而是"在体制规则内找到了所有人都没想到的解法"
**L1 顶层设计**：终点是"在荒诞体制中活出尊严"
**适用**：讽刺/黑色幽默/反乌托邦/比赛/考试题材

### 模式 4：身份弧（五月不行型）
**来源**：《福尔摩斯小姐必不可能成为败犬！》
**结构特征**：
- 以"演出"为单位：策划 → 预告 → 执行 → 意外 → 应对 → 退场 → 结算 → 副作用消化
- 每次演出 = 一条迷你弧（1-3 章）
- 弧的悬念来自"身份是否会暴露"——每次接近暴露时推动关系质变
**L1 顶层设计**：终点是"所有身份的面具都被摘下后，真正的他是谁"
**适用**：多重身份/伪装/怪盗/间谍/侦探题材

### 模式 5：日常累积弧（雨森焚火型）
**来源**：《败犬女主太多了！》
**结构特征**：
- 以"败"（≈ 1/4 卷）为弧单位：日常 → 意外目击/介入 → 被迫互动 → 情感信号泄露 → 回到日常但空气微妙变化
- 弧与弧之间没有明显的危机升级——变化是情感的累积而非事件的递进
- 每卷 4 个"败" + 多个 Intermission（配角视角补充）
**L1 顶层设计**：终点是"从旁观者成为参与者"
**适用**：日常/校园/恋爱喜剧/慢热型关系

### 模式 6：领域隐喻弧（葵关南型）
**来源**：《桌游咖（玩乐关系）》
**结构特征**：
- 每话以"Day XXX"标记——时间是结构骨架而非情节骨架
- 每话引入至少 1 款桌游，该桌游的规则/策略映射本章的关系主题
- 以"秘密管理"为弧的引擎——谁知道了谁的秘密、谁在保护谁的秘密
**L1 顶层设计**：终点是"所有秘密都不再需要"
**适用**：主题型叙事/领域知识型作品/秘密关系/恋爱喜剧

### 结构选择指南

| 你的故事是…… | 推荐结构模式 |
|------------|-----------|
| 世界观本身是最大卖点，读者享受"拼图"过程 | 模式 1 渐进弧 |
| 主角是高智商策略型，读者享受"推演"过程 | 模式 2 目标驱动弧 |
| 故事讽刺某种制度或社会现象 | 模式 3 体制生存弧 |
| 主角有双重/多重身份，悬念来自身份博弈 | 模式 4 身份弧 |
| 日常/情感是核心，事件是关系的"催化剂" | 模式 5 日常累积弧 |
| 主角有某个专业领域的"滤镜"，通过它看世界 | 模式 6 领域隐喻弧 |

## 二、角色原型扩展（C11-C15）

> 原存放于 `framework/templates/characters/_template.md`。以下 5 个原型提炼自参考作品，补充内置的 C1-C10 基础原型。

- **C11 稳健者型**：把"生存"视为最高价值 + 永远有备用方案 + 恐惧不是弱点而是武器。**来源**：言归正传《我师兄实在太稳健了》。**核心特征**：在每次行动前已完成 A/B/C 多方案推演；"被人忽视"是主动策略而非被动状态；内心深处有"为什么我必须活下来"的深层动机。**适用**：修仙/策略/谍战/多势力博弈题材。**常见搭配**：与纯真/热血型角色形成师徒或搭档关系。

- **C12 旁观者型**：自认"路人"/"背景板" + 意外目睹他人私密时刻 + 不情愿的卷入。**来源**：雨森焚火《败犬女主太多了》。**核心特征**：主角自我定位是"配角"，主动与高光角色保持距离；叙事魅力来自"路人视角"下的敏锐观察力——他看到的比别人更多，但选择不发表意见；被迫参与时惯用内心吐槽维持距离。**适用**：日常/校园/轻喜剧/第一人称叙事。**注意**：此原型需要强烈的内心声音——外部沉默但内心滔滔不绝。

- **C13 身份扮演者型**：拥有多重身份/马甲 + 每个身份有明确的行为模式和限制 + 身份切换有"仪式性"标记。**来源**：五月不行《福尔摩斯小姐必不可能成为败犬》。**核心特征**：不是简单的"伪装"，而是每个身份有独立的"人格参数"（语气/姿态/词汇层级/肢体习惯）；身份之间有明确的功能分工；身份泄露是核心戏剧张力的来源。**适用**：怪盗/间谍/侦探/双重生活题材。

- **C14 败犬型**：在感情/竞争中落败 + 不服输但承认败 + 狼狈中见韧性与真实。**来源**：雨森焚火《败犬女主太多了》。**核心特征**：失败不是终点而是角色魅力的起点——"输得起"比"赢"更能赢得读者好感；在失败后的"强撑"姿态中泄露真实情感；有明确的不甘但不会黑化。**适用**：恋爱喜剧中的"炮灰"角色作为主角、竞技/比赛题材中的"亚军叙事"。

- **C15 系统讽刺者型**：身处荒诞体制的最底层 + 保持对体制的清醒认知 + 用吐槽/黑色幽默应对而非革命。**来源**：熊狼狗《没钱修什么仙》。**核心特征**：对"荒诞规则"的适应能力极强——不是认同规则而是寻找生存缝隙；内心吐槽与外在行为的反差是核心笑点来源。**适用**：讽刺/黑色幽默/反乌托邦/"正常人在疯狂世界"题材。

## 四、特殊叙事技法（8 种）

> 原存放于 `framework/templates/style-guide.md` §特殊叙事技法。

### 1. 宏微反差法
**来源**：远瞳 · 《黎明之剑》《深海余烬》
**模式**：在史诗级事件（神战/城市毁灭/文明转折）的收尾处，立刻插入极其日常/幽默的对话或细节。宏大不是为了压垮读者，而是为了衬托角色面对宏大时的"日常态度"。
**核心要领**：宏大的余波必须出现在日常对话中；日常细节必须与之前的宏大事件有逻辑关联；不解释宏大事件的意义——让读者自己从角色的日常态度中感受。
**典型场景**：高潮章收尾、世界规则展示后、角色死亡/离别后、多章弧线结束后

### 2. 认知错位驱动
**来源**：远瞳 · 《黎明之剑》
**模式**：剧情推进不是靠"发生了什么事"，而是靠"角色知道了什么（而别人不知道）"。每个角色的信息都是残缺的，他们的行动基于残缺信息——信息差的交汇点就是剧情爆发点。
**核心要领**：在关键场景前先列出"每个角色知道什么、不知道什么、误解了什么"；读者的信息量应略多于任一角色；认知错位的"纠正"是剧情的高潮。
**典型场景**：多势力政治/博弈、悬疑/推理、喜剧/误解系剧情

### 3. 过度谨慎喜剧
**来源**：言归正传 · 《我师兄实在太稳健了》
**模式**：主角的极度谨慎/过度推演成为喜剧引擎——每个正常事件都被主角当作"可能的风险"过度准备。但当谨慎被证明是"正确的"时，喜剧翻转为角色深度。
**核心要领**：谨慎不能是"被迫害妄想"——每次担忧都必须在逻辑上成立；至少有一次过度谨慎被证明是救命的关键；内心推演的篇幅不能超过行动篇幅。
**典型场景**：多势力博弈、修仙/谍战/策略题材

### 4. 概念置换讽刺
**来源**：熊狼狗 · 《没钱修什么仙》
**模式**：将现代社会现象（贷款/内卷/教育军备）置入奇幻/修仙世界观中——不是简单的"比喻"，而是让荒诞概念在奇幻世界中"字面意义地存在"。
**核心要领**：置换后的概念必须在新世界观中有自洽的内部逻辑；叙述者语气必须保持平静；每个荒诞规则背后必须有现实的"为什么"。
**典型场景**：讽刺/黑色幽默、需要社会批判的奇幻设定

### 5. 多重身份嵌套
**来源**：五月不行 · 《福尔摩斯小姐必不可能成为败犬》
**模式**：角色同时持有多个身份——每个身份有独立的行为模式、语言风格和社交圈。身份切换有"仪式性标志"。核心张力来自"哪个身份才是真正的ta"。
**核心要领**：为每个身份设计独特的"标志动作"和"标志词汇"；身份切换的"仪式"固定且简洁；至少保留一层身份悬念。
**典型场景**：多重身份主角、伪装/卧底/间谍/怪盗、侦探/推理

### 6. 旁观者叙事驱动
**来源**：雨森焚火 · 《败犬女主太多了》
**模式**：主角是"故事的旁观者"而非"故事的英雄"——他目睹他人的情感高潮但不主动参与。叙事魅力来自旁观者的内心声音（观察/吐槽/自我反思）。
**核心要领**：旁观者的内心独白是叙事的第二声轨；旁观者不是无感情的摄像机——他的偏见/误判/迟钝是笑点和情感点的来源；旁观者最终会被迫参与。
**典型场景**：第一人称叙事、日常/校园/轻喜剧

### 7. 双重世界锚点法
**来源**：远瞳 · 《深海余烬》
**模式**：当故事需要在两个世界/两种状态间切换时，设置一个"锚点物品"或"锚点仪式"——每次切换都必须经过它。锚点物同时承担叙事功能（日记=记录+悬念+角色声音）。
**核心要领**：锚点物的每次出现必须推进叙事；两侧世界的变化可以通过"锚点物的变化"来暗示；锚点物的规则一旦建立就不能破坏。
**典型场景**：双重世界/穿越门/现实→异界/时间跳跃

### 8. 仪式化战力释放
**来源**：远瞳 · 《深海余烬》· 舵轮点火；《黎明之剑》· 卫星精视角
**模式**：不在战斗中"顺便"展示能力——每次能力释放有一个独立的"仪式性时刻"。仪式的"步骤感"让读者感受到"这不是普通能力——这是有代价的、有意义的"。
**核心要领**：每次新能力解锁/首次使用必须有独立段落；仪式中必须有感官细节（声音/颜色/温度）；仪式必须有代价/限制。
**典型场景**：能力升级/解锁、关键战斗中的王牌释放、主角"觉醒"时刻

## 五、叙事声音类型（7 种）

> 原存放于 `framework/templates/voice-bible.md` 附录 A。

### A1. 宏微反差声（远瞳型）
**一句话概括**：可以在同一段内从"文明在几万年间兴起与毁灭"切换到"妈蛋，真想仰泳啊"——两者之间不需要过渡。
**核心机制**：外部事件（严肃/史诗）和内心声音（日常/吐槽）同时进行——不是先后切换，而是同一段落内的双声部。
**适用**：奇幻史诗/穿越/克苏鲁/需要"在宏大中保持人性温度"的作品
**参考作品**：《黎明之剑》《深海余烬》《异常生物见闻录》

### A2. 理性推演声（言归正传型）
**一句话概括**：以极其认真、条理分明的语气进行荒诞的过度推演——"师妹来了 → 先检查是不是夺舍 → 再检查有没有妖魔附身 → 再考虑是不是敌人卧底 → 确认了，是正常师妹"。
**核心机制**：主角的内心推演不是"思考描写"而是"叙事引擎"——推演过程 = 信息释放 + 风险揭示 + 角色塑造 + 笑点制造。
**适用**：修仙/策略/谍战/多势力博弈
**参考作品**：《我师兄实在太稳健了》

### A3. 冷静荒诞声（熊狼狗型）
**一句话概括**：用新闻播报般的平静语气叙述疯狂的体制规则。
**核心机制**：叙述者的语气从不配合内容的荒诞程度——越是荒诞的内容越用平静的语气写。反差产生黑色幽默。
**适用**：讽刺/黑色幽默/反乌托邦
**参考作品**：《没钱修什么仙？》

### A4. 优雅狼狈声（五月不行型）
**一句话概括**：月光下以无可挑剔的宫廷礼向警察道晚安 → 凌晨三点因为低血糖差点休克 → 早上因为缺钱蹭室友的橘子酱。
**核心机制**：高光时刻的"优雅"与日常时刻的"狼狈"无缝切换——反差越大，角色魅力越强。
**适用**：多重身份主角/怪盗/间谍/伪装
**参考作品**：《福尔摩斯小姐必不可能成为败犬！》

### A5. 旁观者声（雨森焚火型）
**一句话概括**：像深夜在家庭餐厅，一个嘴硬心软的家伙一边喝着哈密瓜苏打一边跟你讲他今天目睹的事。他不参与——但他讲的每件事都让你觉得"这个人其实很在意"。
**核心机制**：主角以"路人"自居但具有极其敏锐的观察力——他看到的比别人更多，只是选择不发表意见。内心吐槽构成叙事的"评论音轨"。
**适用**：日常/校园/恋爱喜剧/第一人称叙事
**参考作品**：《败犬女主太多了！》

### A6. 末日抒情声（风月型）
**一句话概括**：以诗意的、意象高密度的语言叙述末日——不是在"讲述事件"而是在"描绘一幅画"。
**核心机制**：视觉意象是叙事的第一推动力——每个场景的"画面感"先于"发生了什么"。碎片化的档案/日记/记录嵌入叙事，形成信息差和节奏变化。
**适用**：末世/黑暗奇幻/史诗悲剧
**参考作品**：《天启预报》

### A7. 桌游宅吐槽声（葵关南型）
**一句话概括**：用桌游规则理解一切人际关系——"这个三角关系就是区域占优问题""我的暗恋属于隐藏信息机制"。不是刻意的比喻，而是主角的大脑只能用桌游语言处理社交信息。
**核心机制**：主角有一个极其狭窄但极其深入的知识领域（桌游），他通过这个领域理解整个世界。领域内的术语成为叙事的"基础词汇场"。
**适用**：主题型叙事（以某个领域隐喻人生）、知识型主角
**参考作品**：《桌游咖（玩乐关系）》

### 声音选择指南

| 你想让读者感受到…… | 推荐对标声音 |
|-------------------|------------|
| "这个世界很大很危险，但活着挺好的" | A1 宏微反差声 |
| "这个主角想太多了——但他想得都对" | A2 理性推演声 |
| "这个世界的规则太荒诞了——等等这不就是我们的世界吗" | A3 冷静荒诞声 |
| "这个主角又帅又狼狈——我两个都喜欢" | A4 优雅狼狈声 |
| "我就是个路人——但我看到的东西很重要" | A5 旁观者声 |
| "一切已经结束了——但我还想知道为什么" | A6 末日抒情声 |

### 混合声音指南

- A1 + A2 = 既宏大又有策略深度的奇幻
- A5 + A4 = 日常系身份伪装
- A3 + A1 = 荒诞末世
- 不建议混合：A6 + A2（抒情与推演互相拖慢节奏）、A3 + A5（冷嘲与温吞互相冲淡）

## 六、词汇风格家族（7 种）

> 原存放于 `framework/templates/vocabulary-bank.md` 附录 A。

### 风格家族 1：口语化沉浸型（远瞳系）
**特征**：大量使用口语化连接词和吐槽标记——"说实话""妈蛋""这TM""就很离谱""你跟我说这是……？"
**效果**：读者感觉叙述者就坐在旁边跟他聊天。降低阅读门槛的同时保持信息密度。
**适用场景**：需要"朋友讲故事"般亲近感的叙事

### 风格家族 2：分析冷静型（言归正传系）
**特征**：高频使用推演连接词——"首先/其次/再者""如果A则B""不排除""概率极低但不可忽视""姑且认为"。方案标签化。
**效果**：读者跟随主角的逻辑链——"他在想什么"本身就是阅读乐趣。
**适用场景**：策略/多势力博弈/需要展示主角智商的作品

### 风格家族 3：制度模仿型（熊狼狗系）
**特征**：将现实制度用语（经济/教育/医疗）以"正经语气"置入奇幻世界——"日息""杠杆""培养方案""向道之心"。
**效果**：词汇本身的"正经感"与内容的"荒诞性"形成双重编码。
**适用场景**：讽刺/黑色幽默/世界观批判

### 风格家族 4：双域切换型（五月不行系）
**特征**：词汇在两个完全不同的语域之间切换——优雅场景使用维多利亚英语翻译腔，日常场景使用现代吐槽腔。系统提示用极简精确的短句。
**效果**：语域切换本身就是"身份切换"的信号。
**适用场景**：多重身份/双重生活/伪装叙事

### 风格家族 5：弱表达情感型（雨森焚火系）
**特征**：从不直接说强烈的情感词——"喜欢"被替换为"今天的薯条好像比平时好吃"；"感动"被替换为"我假装在看菜单"。
**效果**：情感的"不表达"比"表达"更有力——读者需要主动解读情感信号，建立更深的情感投入。
**适用场景**：日常/恋爱喜剧/需要"克制表达"的叙事

### 风格家族 6：意象高密型（风月系）
**特征**：高频使用自然/宇宙意象词汇——冰/火/灰/光/影/星辰/海洋/废墟/钟声。偏古典书面语，不使用口语化连接词。
**效果**：每个句子都有视觉重量。意象之间形成隐喻网络。
**适用场景**：史诗/末世/悲剧/需要"文学重量"的作品

### 风格家族 7：领域沉浸型（葵关南系）
**特征**：主角的认知框架完全浸泡在某个特定领域（桌游/音乐/体育/科学）中，该领域的术语成为叙事的基础词汇场。离开这个领域的词汇，主角就不知道如何描述世界。
**效果**：叙事的独特性来自领域的独特性——读者在学领域知识的同时也理解了主角这个人。
**适用场景**：知识型主角、主题型作品（以某个领域隐喻人生）
