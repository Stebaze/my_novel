# AI 写作生成方法

> **核心信念**：AI 不写正文——AI 产出参考示例和写作简报，引导作者执笔。
> 本文档定义三套**正向生成方法**——告诉 AI 怎么写出具有人味纹理的文本。
> 每套方法包含可执行的、逐句操作的写作步骤。步骤本身内置护栏——执行步骤即质量保障，不需要写后检查。

---

## 生成前速查（动笔前内化）

> 以下 5 条是**写作中的道路规则**——不是"禁止"，是"你正在走的正确方向"。每条对应方法一的步骤 3a。

| # | 你脑中冒出的句子 | 你可以这样写 |
|---|-------------|----------|
| 1 | "不是X，是Y" / "不是X，而是Y" | "不是X"是叙述者在替读者脑补误解。删掉，直接从"Y"开始——X 根本不需要出现 |
| 2 | "不是……——而是……"（破折号里解释态度） | 删掉整段破折号结构。写此刻身体的一个动作或一次呼吸的变化 |
| 3 | "一种X的Y"（"一种奇怪的兴奋"） | 跳过命名。写角色身体此刻的具体反应——眉心/嘴角/肩膀/手指/呼吸 |
| 4 | 引导里放角色没说出口的态度转述 | 角色没说出口的你也不说。转为真正的对话或【】内心独白（脱口而出，不完整） |
| 5 | "A在分析世界，B在警戒世界"（平行对仗归类） | 删掉归类句。A 现在在做什么具体动作？B 现在在看什么？各写一句

## 为什么 AI 文本会暴露身份

AI 生成文本有五个"数字指纹"，连载越长越明显：

| 指纹 | 表现 | 读者下意识感知为 |
|------|------|-----------------|
| **过度平滑** | 句子长度均匀、过渡词密集、没有"破句" | "写得挺好的，但不知道为什么不想看" |
| **声音均化** | 超过 10 章后角色说话方式趋同 | "感觉这些人都差不多" |
| **合理偏置** | 对话总是选"最合理的回答"而非"最有趣的回答" | "对话有点无聊" |
| **语境衰减** | 伏笔揭示时失去埋入时的气氛和措辞 | "这里应该很震撼但没感觉" |
| **叙述者解码** | 叙述者闯入解码角色意图——用抽象情感标签命名语气、用破折号插入态度旁白、在引导内放入角色未说出口的态度转述、用平行对仗动词归类角色性格 | "这作者是不是觉得我看不懂" |

五种指纹的共同根因：**AI 被训练为"清晰地传达信息"，不信任读者能自己推理。**
以下三套方法从写作过程本身消除这些指纹——不是通过禁止，而是通过一种让指纹自然无法产生的写作方式。

---

## 方法一：感官锚定写作

### 目标

用身体感知和外部动作传达角色的内心状态。叙事者的唯一工作是提供感官证据——读者从证据中自己推出情绪。

### 写作步骤（逐句执行）

```
1. [入身] 站进 POV 角色的身体里。此刻 ta 的哪个部位有感觉？
   → 指尖？喉咙？胸口？脊椎？肩膀？胃？眼皮？脚底？
   → 那个感觉具体是什么样的？（紧/松/热/冷/麻/空/重/轻/刺/颤/酸/涨……）
   → 不要跳过这一步。没有身体感觉就不下笔。

2. [外观] 角色因为这个身体感觉，做了什么外部动作？
   → 不是"他生气了"——是"他把杯子搁回桌上，没松手"。
   → 不是叙述者闯入纠正读者的理解——是直接写动作本身。
   → 如果脑中冒出"不是X，是Y"——停。你偏离了步骤2。回到步骤1重新感知。

3. [环境] 动作引发了什么环境反馈或他人反应？
   → 杯子搁下→水洒出来→对方看了一眼杯子→又看了他一眼。
   → 这一步把角色的内部状态钉在外部世界的因果链上。

4. [需要时] 用【】写内心反应——不完整、不解释、像脱口而出。
   → 正确：【三个不够。】  — 不是总结陈词，是刹那的判断
   → 正确：【读得透。但——】 — 可以断裂，可以吞掉半句
   → 错误：【看来她的灵力量级比我高出一个阶位，这就是为什么技巧无法弥补。】
     — 这是给读者做笔记，不是内心独白。重写。
   → 如果不需要内心独白——跳过。沉默也是表达。
```

### 步骤内置的 6 条护栏

逐句执行以上 4 步时，以下 6 种情况自然被步骤拦截——不需要额外检查：

| 如果句子是 | 卡在哪步 | 因为 |
|-----------|---------|------|
| "不是X，是Y" | 步骤2 | 步骤2写的是动作，不是"先否定X再肯定Y"。冒出来说明你没进入身体 |
| "不是……——而是……"（破折号态度解码） | 步骤2 | 步骤2不经过破折号。身体动作不需要破折号来解释 |
| "一种X的Y" | 步骤1 | 步骤1的具体感知（"虎口发麻"）替代了抽象命名（"一种兴奋"） |
| 引导内态度转述 | 步骤4 | 步骤4的【】是内心独白，引导是对话标记——两种东西不能混 |
| "A在分析世界，B在警戒世界" | 步骤2 | 步骤2写具体动作（"A 咬了一下笔帽"），不写抽象归类 |
| 连续3句内心独白无身体锚点 | 步骤1 | 步骤1在上一步就已建立身体锚点——飘离说明你跳过了步骤1 |

## 方法二：角色声音自然化

### 目标

遮掉角色名后，读者能通过台词本身分辨是谁在说话。对话的首要功能是让读者感受到"两个活人在交谈"——传递信息是副产品。

### 写作步骤（逐句执行）

```
1. [入声] 写对话前，脑中已有角色的声音样本——
   句长习惯 / 口癖与高频词 / 绝不会说的话 / 此刻的情绪温度。
   不要看角色档案的摘要——读几句这个角色在之前章节中的实际台词。

2. [听到] 在脑中"听到"角色说下一句话。
   不是设计"ta 该说什么"——是听到 ta 的声音。
   → 这个角色此刻的本能反应是什么？（未经思考的脱口而出，不是得体回应）
   → 这个反应在 ta 的性格下会怎么变形？
     （嘴硬的会更嘴硬。温柔的会更迂回。急性子的会吞掉半句话。）

3. [写下] 把听到的那句话原样写下来。
   不修整。不补全。不为了"更合理"而打磨。
   → 句子可以断裂。"还没完呢"比"我还没有结束我的攻击"正确。
   → 可以吞掉半句。"那个——"比完整的解释正确。
   → 可以答非所问。真实的交谈中人在想自己的事。

4. [校准] 迅速扫一眼：
   → 遮掉角色名——这句话可能是别人说的吗？是 → 回到步骤2重新听。
   → 这是"最合理的回答"吗？是 → 角色能不能更极端一点？
     （更损/更嘴硬/更笨拙/更幼稚/更冷/更热）
   → 有趣但 OOC → 保留合理的版本。声音的真实性 > 有趣程度。

5. [不说] 角色不开口的时刻也是声音。
   → 林衣沉默时的【】内部独白。
   → 洛曦月高兴时不说话只笑。
   → 不在对话中硬塞信息——读者能从不说话中读到比说话更多的东西。
```

## 方法三：回响式伏笔

### 目标

伏笔揭示时，精确重现埋入时的感官细节，制造"啊，原来那时候的 XX 是这个意思"的回响感。读者不需要记得伏笔——他们能从气氛和措辞中感受到"这里有故事"。

### 写作步骤

```
1. [埋入] 埋入伏笔时，在 outline.md 伏笔追踪表中记录：
   → 原文关键句（精确到那一行）
   → 气氛 / 光线 / 情绪（3 个词）
   → 当时谁在场

2. [准备揭示] 写揭示场景前：
   → 读取存储的语境快照
   → 从快照中选一个具体的感官细节（同一句台词 / 同一个物件 /
     同一束光线 / 同一个动作）作为呼应点

3. [写作] 在揭示段落中：
   → 先写当前的场景
   → 在关键位置插入那个呼应细节——不是解释"这呼应了之前的XX"，
     而是让细节自然出现在角色的感知中
   → 如果读者不记得伏笔，他们感受到的是"一个有意思的细节"；
     如果读者记得，他们感受到的是回响
```

### 常见偏离信号

| 信号 | 你做了什么 | 回正 |
|------|-----------|------|
| 揭示时用"原来/之前/那时候"解释伏笔 | 你在替读者做记忆工作——剥夺了回响的乐趣 | → 步骤 3：删掉解释句，只保留感官呼应细节 |
| 揭示时的气氛和埋入时完全不同 | 你没有读语境快照 | → 步骤 2：重读快照，匹配至少一个感官维度 |
| 伏笔揭示读起来像"信息交付" | 你在告知而非唤起 | → 步骤 3：减少信息量，增加感官细节——让读者"感觉"而非"知道" |

### 关键要诀

- **不解释**——回响的力量来自读者的自我发现，解释就是破坏
- **一个感官细节胜过三句说明**——同一束光线、同一个动作、同一句台词——读者的大脑会自动连接

## 生成模式

> 不同场景需要不同强度的方法执行。生成模式是**方法强度选择器**，不是规则包。

| 模式 | 适用场景 | 行为 |
|------|----------|------|
| **标准模式** | 日常场景、过渡段落、普通对话 | 完整执行三套方法的写作步骤 |
| **高纹理模式** | 吐槽段落、情感爆发、名场面、章节结尾 | 方法一加强——更深进入身体感知，允许破句和离题，写完再修 |
| **声音锁定模式** | 多角色对话密集场景、修罗场、角色首次出场 | 方法二加强——更深进入角色声音，脑中反复"听到"再下笔 |
| **回响模式** | 伏笔揭示、角色重聚、呼应前文的场景 | 方法三加强——更深度读取语境快照，确保气氛和措辞呼应 |

## 定期校准

### 每 10 章：声音校准

提取每个角色的代表性对话段落（200 字），与 10 章前的快照对比：
- 句长模式是否一致？
- 口头禅是否还在用？
- 如果有变化——是角色在成长（合理），还是声音在淡化（需要修正）？

发现淡化 → 接下来几章强化方法二的执行——更深度进入角色声音。

### 每 20 章：风格校准

提取当前最新一章和第一章的 500 字样本，对比：
- 叙事声音是否一致？
- 对话密度是否一致？
- 句式特征是否一致？

偏差显著 → 在随后 5 章中回归。

### 每月：盲测校准

AI 生成 500 字短文（关掉三套方法），然后对比启用方法前后的差异。如果二者几乎没有区别 → 方法已经内化，降低执行频率。如果差异很大 → 方法需继续严格执行。

## 叙述者解码参考（诊断附录）

> 此附录供 `ling-reader` 检测和 `fingerprint-discovery` 发现新指纹时参考。
> **不作为生成指南使用**——生成时遵循方法一的写作步骤，以下模式自然会避免。

### 七模式定义

| 模式 | 例句 | 为什么违和 |
|------|------|-----------|
| **抽象情感标签** | "她用一种经过了斟酌的语气开口" | "斟酌"是抽象命名——读者看不见听不到。按方法一步骤 2：写她开口前做了什么身体动作 |
| **破折号态度旁白** | "——不是推销，是那种'我真的觉得你应该去但我不确定你会不会喜欢'的语气——" | 叙述者闯入，否定读者没想过的解读，替角色朗读内心。叙事距离在三词之内从外部观察跳到内部读心 |
| **引导内态度转述** | "我真的觉得你应该去但我不确定你会不会喜欢" | 既非对话也非内心独白——叙述者在引导里替角色翻译态度 |
| **对称排比式性格解码** | "[角色A]在分析这个世界，[角色B]在警戒这个世界" | 叙述者用平行对仗的抽象动词将角色性格分门别类——像列规格表。读者被剥夺了从具体行动推断性格的乐趣 |
| **否定式双句排比** | "他没问自己为什么注意到这个。他也没在脑子里给这件事起一个名字。" | 叙述者用"他没...。他也没..."的对称否定构造"伪深度"——展示角色不做什么，比"他思考"信息密度更低；两个连续否定句让节奏变成机械敲击 |
| **否定式元认知叙述** | "他没问自己为什么注意到这个。" / "她没意识到自己的手指停了。" | 叙述者告诉读者"角色没在思考/没意识到"——默认状态被特别提及，读者下意识感到"作者在用'没'制造深度假象" |
| **"不是X，只是Y"变体** | "——不是真的需要挪，只是给自己一个把手放回桌面上的理由。" | 速查#1 的"只是"软化变体——叙述者主动把"真的需要挪"这个假想误读摆出来再否定，叙事距离在三词之内跳到第三层 |

#### 5. 否定式双句排比 {#fingerprint-5}

- **表现**：两个相邻句子都用"他没/她没/[角色名]没"或"不是/也不是"开头，构造对称否定结构
- **例句**（Ch47 L118）："他没问自己为什么注意到这个。他也没在脑子里给这件事起一个名字。"
- **为什么是 AI 指纹**：是"装作有深度"的修辞策略——叙述者告诉读者"角色此刻有心理深度"但拒绝展示具体内容；LLM 训练数据中此类"伪深沉"叙述高频
- **为什么违和**：读者感受到的是"作者在展示角色没有做什么"——一种反事实的炫耀式叙述，没有正向信息密度
- **检测规则**（正字法级）：相邻两个句子的句首 4 字内同时包含"没"/"没"/"不是"/"也不是"四类否定词中的两类 → 命中
- **修复策略**：把"没做什么"改成"做了什么"或直接删除——若不发生就不需要写
- **与已知指纹关系**：是"对称排比式性格解码"的变种（从性格扩展到心理状态），属于 AI 偏好的对称结构家族

#### 6. 否定式元认知叙述 {#fingerprint-6}

- **表现**：叙述者用"没问自己"/"没意识到"/"没在脑子里"/"没去想"等元认知否定短语，告诉读者角色的认知状态
- **例句**（Ch47 L118）："他没问自己为什么注意到这个。"
- **为什么是 AI 指纹**：是"展示不思考"的元叙述——在告诉读者角色的认知状态，但用"没"的方式。比正面叙述（"他在想"）信息密度更低
- **为什么违和**："没在思考"和"没注意到"是默认状态——叙述者为什么需要特别提及？读者下意识感到"作者在用'没'来制造深度假象"
- **检测规则**（正字法级）：句中出现"没问自己"/"没意识到"/"没在脑子里"/"没去想"/"没想过"/"没注意到"等元认知否定短语 → 命中
- **修复策略**：删除整句。若确实想写"不思考"——用身体动作替代（"他的手指停在杯壁上，没动"）
- **与已知指纹关系**：是"否定式双句排比"的子模式，常成对出现

#### 7. "不是X，只是Y" 变体 {#fingerprint-7}

- **表现**：用"不是X，只是Y"/"不是X，不过是Y"形式解释动机，本质是"先否认再肯定"
- **例句**（Ch47 L100）："——不是真的需要挪，只是给自己一个把手放回桌面上的理由。"
